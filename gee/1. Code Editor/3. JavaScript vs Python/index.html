    JavaScript vs Python: What's the Difference?¶

    Google Earth Engine works with two languages: JavaScript and Python. They do the exact same things in GEE, but they look slightly different.

    Quick Comparison¶

    Feature JavaScript (Code Editor) Python (Jupyter/Colab)
    Variables var image = ... image = ... (No var)
    Functions function myFunc(x) { ... } def my_func(x):
    Comments // This is a comment # This is a comment
    True/False true, false (lowercase) True, False (Capitalized)
    Null null None
    Dictionaries {key: 'value'} {'key': 'value'} (Quotes on keys)
    Install? No (Runs in browser) Yes (Requires setup)

    Code Examples¶

    Here is how common tasks look in both languages.

    1. Loading an Image¶

    // Load Landsat 8
var image = ee.Image('LANDSAT/LC08/C02/T1_L2/LC08_044034_20140318');

print('Image loaded:', image);

    # Load Landsat 8
image = ee.Image('LANDSAT/LC08/C02/T1_L2/LC08_044034_20140318')

print('Image loaded:', image.getInfo()) # Note: .getInfo() needed to print

    2. Defining Visualization Parameters¶

    This is the biggest difference! Python requires quotes around keys in dictionaries.

    var visParams = {
  bands: ['SR_B4', 'SR_B3', 'SR_B2'],
  min: 7000,
  max: 12000,
  palette: ['blue', 'green', 'red']
};

Map.addLayer(image, visParams, 'My Image');

    vis_params = {
    'bands': ['SR_B4', 'SR_B3', 'SR_B2'],
    'min': 7000,
    'max': 12000,
    'palette': ['blue', 'green', 'red']
}

Map.addLayer(image, vis_params, 'My Image')

    3. Writing a Function¶

    // A function to mask clouds
var maskClouds = function(image) {
  var qa = image.select('QA_PIXEL');
  // ... logic here ...
  return image.updateMask(qa);
};

// Apply it using .map()
var cleanCollection = collection.map(maskClouds);

    # A function to mask clouds
def mask_clouds(image):
  qa = image.select('QA_PIXEL')
  # ... logic here ...
  return image.updateMask(qa)

# Apply it using .map()
clean_collection = collection.map(mask_clouds)

    Which one should I use?¶

    • Choose JavaScript if: You are a beginner. It runs instantly in the browser and is easiest to learn.
    • Choose Python if: You are a data scientist, want to use other libraries (like pandas/matplotlib), or want to automate workflows.
