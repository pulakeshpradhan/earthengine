## Google Earth Engine: JavaScript vs. Python Syntax Comparison

Example: Basic Image Loading
JavaScript (GEE)
```javascript
// Load an image from the Landsat 8 collection
var image = ee.Image('LANDSAT/LC08/C01/T1_SR/LC08_044034_20140318');

// Select the NIR band
var nir = image.select('B5');

// Display on the map
Map.centerObject(image, 9);
Map.addLayer(nir, {min: 0, max: 3000, palette: ['blue', 'green', 'red']}, 'NIR Band');
```
Python (GEE)

```python
# Load an image from the Landsat 8 collection
image = ee.Image('LANDSAT/LC08/C01/T1_SR/LC08_044034_20140318')
```

Example: Creating a Custom Function

```javascript
// Function to mask clouds in Landsat 8 images
var maskClouds = function(image) {
    var cloudShadowBitMask = (1 << 3);
    var cloudsBitMask = (1 << 5);
    var qa = image.select('pixel_qa');
    var mask = qa.bitwiseAnd(cloudShadowBitMask).eq(0).and(
                qa.bitwiseAnd(cloudsBitMask).eq(0));
    return image.updateMask(mask);
};
```
Python (GEE)
```python
# Function to mask clouds in Landsat 8 images
def mask_clouds(image):
    cloud_shadow_bit_mask = (1 << 3)
    clouds_bit_mask = (1 << 5)
    qa = image.select('pixel_qa')
    mask = qa.bitwiseAnd(cloud_shadow_bit_mask).eq(0).And(
           qa.bitwiseAnd(clouds_bit_mask).eq(0))
    return image.updateMask(mask)
```

Example: Reducing an Image Collection with a Custom Reducer

```javascript
// Custom reducer function to calculate NDVI mean and standard deviation
var customReducer = function(collection) {
    var mean = collection.select('NDVI').mean();
    var stdDev = collection.select('NDVI').reduce(ee.Reducer.stdDev());
    return mean.addBands(stdDev);
};
```
Python (GEE)
```python
# Custom reducer function to calculate NDVI mean and standard deviation
def custom_reducer(collection):
    mean = collection.select('NDVI').mean()
    std_dev = collection.select('NDVI').reduce(ee.Reducer.stdDev())
    return mean.addBands(std_dev)
```
| JavaScript (GEE)            | Python (GEE)                   | Notes                                          |
|-----------------------------|--------------------------------|------------------------------------------------|
| `var`                       | No `var`, just assign variable | Declare variables directly in Python           |
| `function`                  | `def`                          | Define functions using `def`                   |
| `{ ... }`                   | Indentation with `:`           | Replace `{}` with `:` and indent               |
| `;`                         | No semicolons                  | Python syntax omits semicolons                 |
| `//`                        | `#`                            | Use `#` for comments instead of `//`           |
| Inline function             | `lambda`                       | Use `lambda` for inline functions in Python    |
| Visualization dictionary `{}` | Dictionary `{...}` in Python  | Use dictionaries directly in visualization params |
