# Google Earth Engine Code Editor Shortcuts

## General Editing Shortcuts

| Shortcut                       | Action                                   |
|--------------------------------|------------------------------------------|
| `Ctrl + /` (or `Cmd + /`)      | Toggle comment/uncomment on selected lines |
| `Ctrl + Z` (or `Cmd + Z`)      | Undo                                     |
| `Ctrl + Shift + Z` (or `Cmd + Shift + Z`) | Redo                   |
| `Ctrl + C` (or `Cmd + C`)      | Copy selected text                       |
| `Ctrl + X` (or `Cmd + X`)      | Cut selected text                        |
| `Ctrl + V` (or `Cmd + V`)      | Paste                                    |
| `Ctrl + F` (or `Cmd + F`)      | Find in code                             |
| `Ctrl + Shift + F` (or `Cmd + Shift + F`) | Find and replace         |
| `Tab`                          | Indent selected line(s)                  |
| `Shift + Tab`                  | Unindent selected line(s)                |

## Code Navigation Shortcuts

| Shortcut                       | Action                                   |
|--------------------------------|------------------------------------------|
| `Ctrl + Space`                 | Trigger code autocomplete                |
| `Ctrl + Shift + D`             | Duplicate current line or selection      |
| `Ctrl + [Arrow Up/Down]`       | Move line(s) up or down                  |
| `Ctrl + Home`                  | Go to the top of the script              |
| `Ctrl + End`                   | Go to the end of the script              |
| `Alt + [Arrow Up/Down]`        | Navigate up/down through multi-line selections |

## Console & Output Shortcuts

| Shortcut                       | Action                                   |
|--------------------------------|------------------------------------------|
| `Ctrl + Enter`                 | Run the script                           |
| `Ctrl + Shift + Enter`         | Save the script and run it               |
| `Esc`                          | Close the current dialog (if any)        |
| `Shift + Click on Console Panel` | Toggle console open/close            |

## Map Shortcuts

| Shortcut                       | Action                                   |
|--------------------------------|------------------------------------------|
| `Shift + Scroll`               | Zoom in/out                              |
| `Shift + Click + Drag`         | Pan map                                  |
| `Double Click on Map`          | Zoom in at clicked location              |

## Help & Documentation Shortcuts

| Shortcut                       | Action                                   |
|--------------------------------|------------------------------------------|
| `Ctrl + Shift + /`             | Open documentation search                |
| `Ctrl + Shift + P`             | Quick command palette                    |
