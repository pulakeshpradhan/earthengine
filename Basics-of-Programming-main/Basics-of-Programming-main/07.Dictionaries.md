# Dictionaries in Google Earth Engine
=====================================

Dictionaries in Google Earth Engine (GEE) are key-value pair data structures used to store and organize data. They are implemented as `ee.Dictionary` objects and are similar to JavaScript objects or Python dictionaries.


## Creating Dictionaries
----------------------

### 1. Using `ee.Dictionary()`

```javascript
var dict = ee.Dictionary({
  'name': 'John Doe',
  'age': 30,
  'city': 'New York'
});
print('Dictionary:', dict);
```
### 2. From a List of Keys and Values
```JavaScript
var keys = ['a', 'b', 'c'];
var values = [1, 2, 3];
var dictFromLists = ee.Dictionary.fromLists(keys, values);
print('Dictionary from lists:', dictFromLists);
``` 
### Accessing Dictionary Values
-----------------------------
### 1. Using get()
```JavaScript
var name = dict.get('name');
print('Name:', name);
```
###2. Using getNumber(), getString(), etc.
```JavaScript
var age = dict.getNumber('age');
print('Age as number:', age);
```
### Modifying Dictionaries
------------------------
Dictionaries in GEE are immutable, but you can create new dictionaries with modifications.
1. Adding or Updating Key-Value Pairs
```JavaScript
var updatedDict = dict.set('occupation', 'Engineer');
print('Updated Dictionary:', updatedDict);
```
2. Removing Key-Value Pairs
```JavaScript
var reducedDict = dict.remove(['age']);
print('Dictionary with age removed:', reducedDict);
```
### Dictionary Operations
-----------------------
1. Combining Dictionaries
```JavaScript
var dict1 = ee.Dictionary({'a': 1, 'b': 2});
var dict2 = ee.Dictionary({'c': 3, 'd': 4});
var combinedDict = dict1.combine(dict2);
print('Combined Dictionary:', combinedDict);
```
2. Checking for Keys
```JavaScript
var hasKey = dict.contains('name');
print('Dictionary contains "name":', hasKey);
```
3. Getting All Keys or Values
```JavaScript
var allKeys = dict.keys();
var allValues = dict.values();
print('All Keys:', allKeys);
print('All Values:', allValues);
```
### Advanced Usage
-----------------
1. Mapping Over Dictionary Values
``` JavaScript
var doubledValues = dict.map(function(key, value) {
  return ee.Number(value).multiply(2);
});
print('Doubled numeric values:', doubledValues);
````
2. Using Dictionaries with Image Properties
``` JavaScript
var image = ee.Image('LANDSAT/LC08/C02/T1_L2/LC09_044034_20230215')
  .set({
    'cloud_cover': 10,
    'sensor': 'OLI',
    'acquisition_date': '2023-02-15'
  });

var properties = image.toDictionary();
print('Image properties as dictionary:', properties);
```
### 3. Creating a FeatureCollection from a Dictionary
``` JavaScript
var dictFeature = ee.Feature(null, dict);
var dictFeatureCollection = ee.FeatureCollection([dictFeature]);
print('FeatureCollection from Dictionary:', dictFeatureCollection);
```
### Best Practices
-----------------
1. Use Descriptive Key Names
Use descriptive key names for clarity.
2. Immutability
Be aware of the immutability of dictionaries in GEE.
3. Getter Methods
Use appropriate getter methods (getString(), getNumber(), etc.) when you know the value type.
4. Organizing Metadata
Leverage dictionaries for organizing metadata and properties in your Earth Engine workflows.
Conclusion
----------
Dictionaries in Google Earth Engine are versatile data structures that can be used in various scenarios, from simple data organization to complex image and feature property management.
