# স্থানিক বিশ্লেষক (Spatial Analyst) — বাংলা সারসংক্ষেপ

> **নোট:** এই ফাইলটি `Spatial-Analyst.md` (৪৫KB) এর বাংলা সারসংক্ষেপ। সম্পূর্ণ বিস্তারিত বিবরণের জন্য মূল ইংরেজি ফাইল দেখুন।

---

## মূল বিষয়সমূহ

### ১. র‍্যাস্টার ডেটা বিশ্লেষণ

- পিক্সেল-ভিত্তিক গণনা ও ইমেজ অ্যালজেব্রা
- জোনাল পরিসংখ্যান (Zonal Statistics)
- রিক্লাসিফিকেশন

### ২. ভেক্টর ডেটা বিশ্লেষণ

- বাফার বিশ্লেষণ
- ওভারলে অপারেশন (ইন্টারসেক্ট, ইউনিয়ন, ক্লিপ)
- প্রক্সিমিটি বিশ্লেষণ

### ৩. ইন্টারপোলেশন

- IDW (Inverse Distance Weighting)
- ক্রিগিং (Kriging)
- স্প্লাইন ইন্টারপোলেশন

### ৪. ভূমিরূপ বিশ্লেষণ

- ঢাল (Slope)
- দিক (Aspect)
- হিলশেড (Hillshade)
- কনট্যুর তৈরি

### ৫. জলবিদ্যা বিশ্লেষণ

- প্রবাহ দিক (Flow Direction)
- প্রবাহ সঞ্চয় (Flow Accumulation)
- জলনিষ্কাশন অববাহিকা (Watershed Delineation)

### ৬. দূরত্ব বিশ্লেষণ

- ইউক্লিডীয় দূরত্ব
- খরচ দূরত্ব (Cost Distance)
- সবচেয়ে কম খরচের পথ

### ৭. ঘনত্ব বিশ্লেষণ

- কার্নেল ঘনত্ব অনুমান (KDE)
- পয়েন্ট ডেনসিটি

---

## মূল টুলসমূহ

| টুল | উদ্দেশ্য |
| :--- | :--- |
| **Zonal Statistics** | নির্দিষ্ট জোনে পিক্সেল মান সংক্ষিপ্ত করা |
| **Buffer** | ফিচারের চারপাশে নির্দিষ্ট দূরত্বের জোন তৈরি |
| **Overlay** | একাধিক লেয়ারের স্থানিক সম্পর্ক বিশ্লেষণ |
| **Reclassify** | পিক্সেল মানকে নতুন শ্রেণীতে পুনর্শ্রেণীবদ্ধ করা |
| **Terrain** | DEM থেকে ঢাল, দিক ইত্যাদি গণনা |
