# গুগল আর্থ ইঞ্জিন কোড এডিটর শর্টকাট (Keyboard Shortcuts)

## সাধারণ এডিটিং শর্টকাট

| শর্টকাট | কাজ |
|--------------------------------|------------------------------------------|
| `Ctrl + /` (বা `Cmd + /`) | নির্বাচিত লাইনগুলোতে কমেন্ট টগল করুন |
| `Ctrl + Z` (বা `Cmd + Z`) | আনডু (Undo) |
| `Ctrl + Shift + Z` (বা `Cmd + Shift + Z`) | রিডু (Redo) |
| `Ctrl + C` (বা `Cmd + C`) | নির্বাচিত টেক্সট কপি করুন |
| `Ctrl + X` (বা `Cmd + X`) | নির্বাচিত টেক্সট কাট করুন |
| `Ctrl + V` (বা `Cmd + V`) | পেস্ট করুন |
| `Ctrl + F` (বা `Cmd + F`) | কোডে খুঁজুন |
| `Ctrl + Shift + F` (বা `Cmd + Shift + F`) | খুঁজুন এবং প্রতিস্থাপন করুন |
| `Tab` | নির্বাচিত লাইনে ইনডেন্ট দিন |
| `Shift + Tab` | ইনডেন্ট সরান |

## কোড নেভিগেশন শর্টকাট

| শর্টকাট | কাজ |
|--------------------------------|------------------------------------------|
| `Ctrl + Space` | কোড অটোকমপ্লিট |
| `Ctrl + Shift + D` | বর্তমান লাইন বা সিলেকশন ডুপ্লিকেট করুন |
| `Ctrl + [Arrow Up/Down]` | লাইন ওপরে/নিচে সরান |
| `Ctrl + Home` | স্ক্রিপ্টের শুরুতে যান |
| `Ctrl + End` | স্ক্রিপ্টের শেষে যান |

## কনসোল ও আউটপুট শর্টকাট

| শর্টকাট | কাজ |
|--------------------------------|------------------------------------------|
| `Ctrl + Enter` | স্ক্রিপ্ট রান করুন |
| `Ctrl + Shift + Enter` | স্ক্রিপ্ট সেভ করুন এবং রান করুন |
| `Esc` | বর্তমান ডায়ালগ বন্ধ করুন |

## ম্যাপ শর্টকাট

| শর্টকাট | কাজ |
|--------------------------------|------------------------------------------|
| `Shift + Scroll` | জুম ইন/আউট |
| `Shift + Click + Drag` | ম্যাপ প্যান করুন |
| `Double Click` | ক্লিক করা জায়গায় জুম ইন করুন |

## সাহায্য ও ডকুমেন্টেশন শর্টকাট

| শর্টকাট | কাজ |
|--------------------------------|------------------------------------------|
| `Ctrl + Shift + /` | ডকুমেন্টেশন সার্চ খুলুন |
| `Ctrl + Shift + P` | কুইক কমান্ড প্যালেট |
