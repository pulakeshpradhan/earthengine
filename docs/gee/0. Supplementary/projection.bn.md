# আর্থ ইঞ্জিনে প্রজেকশন (বাংলা সারসংক্ষেপ)

> **নোট:** এই ফাইলটি `projection.md` (৮২KB) এর বাংলা সারসংক্ষেপ। সম্পূর্ণ বিস্তারিত বিবরণের জন্য মূল ইংরেজি ফাইল দেখুন।

---

## মূল বিষয়সমূহ

### প্রজেকশনের মিলবিদ্যা

প্রজেকশন হলো ত্রিমাত্রিক পৃথিবীকে সমতল পৃষ্ঠে রূপান্তরের গাণিতিক পদ্ধতি। বিভিন্ন প্রজেকশন বিভিন্ন বৈশিষ্ট্য সংরক্ষণ করে:

| প্রজেকশন টাইপ | সংরক্ষণ করে | বিকৃত করে |
| :--- | :--- | :--- |
| **Conformal** (যেমন- Mercator) | আকৃতি | ক্ষেত্রফল |
| **Equal-area** (যেমন- Albers) | ক্ষেত্রফল | আকৃতি |
| **Equidistant** | দূরত্ব | ক্ষেত্রফল/আকৃতি |

### আর্থ ইঞ্জিনে CRS

- **EPSG:4326** — WGS84, ভৌগোলিক (ডিগ্রি)
- **EPSG:3857** — Web Mercator, ওয়েব মানচিত্রে ব্যবহৃত
- **UTM জোন** — আঞ্চলিক নির্ভুলতার জন্য

### প্রজেকশন তথ্য পরীক্ষা

```javascript
var image = ee.Image('LANDSAT/LC08/C02/T1_TOA/LC08_044034_20140318').select(0);
print('প্রজেকশন:', image.projection());
print('মিটারে স্কেল:', image.projection().nominalScale());
```

### কম্পোজিট ও প্রজেকশন

```javascript
var collection = ee.ImageCollection('LANDSAT/LC08/C02/T1_TOA');
var mosaic = collection.filterDate('2018-01-01', '2019-01-01').mosaic();
print(mosaic.projection()); // ডিফল্ট WGS84
```

### রিপ্রজেকশন

```javascript
var proj = ee.Projection('EPSG:4326');
var output = image.reproject(proj);
```

## ⚠️ সতর্কতা

- `reproject()` অতিরিক্ত ব্যবহারে কর্মক্ষমতা কমে
- বিভিন্ন প্রজেকশনের ইমেজ একত্রিত করলে সাবধান থাকুন
- ক্ষেত্রফল গণনায় সঠিক প্রজেকশন ব্যবহার করুন
