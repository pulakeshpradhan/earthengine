# আপনার ডেটা এক্সপোর্ট করা (Exporting Your Data)

ডেটা বিশ্লেষণ বা ম্যাপ তৈরি করার পর আপনি সেটি সংরক্ষণ করতে চাইবেন!
আপনি ফলাফল এক্সপোর্ট করতে পারেন **গুগল ড্রাইভে (Google Drive)**, **গুগল ক্লাউড স্টোরেজে (Google Cloud Storage)** বা **আর্থ ইঞ্জিন অ্যাসেট (Earth Engine Asset)** হিসেবে।

## ১. ইমেজ এক্সপোর্ট করা (GeoTIFF)

একটি ম্যাপ (যেমন- NDVI ইমেজ) গুগল ড্রাইভে ডাউনলোড করতে:

=== "JavaScript"
    ```javascript
    // আপনি যে ইমেজটি এক্সপোর্ট করতে চান সেটি নির্বাচন করুন (যেমন- NDVI)
    var imageToExport = easyNDVI;

    // এক্সপোর্ট প্যারামিটার ডিফাইন করুন
    Export.image.toDrive({
      image: imageToExport,
      description: 'NDVI_Export_2023',
      scale: 30, // রেজোলিউশন মিটারে (ল্যান্ডস্যাটের জন্য ৩০ মিটার)
      region: roi, // যে এলাকাটি এক্সপোর্ট করতে হবে
      fileFormat: 'GeoTIFF'
    });
    ```
    *ডানদিকে "Tasks" ট্যাবে একটি নতুন টাস্ক দেখা যাবে। ডাউনলোড শুরু করতে "Run" ক্লিক করুন!*

=== "Python"
    ```python
    import geemap

    # geemap ব্যবহার করে এক্সপোর্ট করুন (সবচেয়ে সহজ উপায়)
    geemap.ee_export_image_to_drive(
        easy_ndvi, 
        description='NDVI_Export_2023', 
        folder='GEE_Exports', 
        scale=30, 
        region=roi
    )
    ```

## ২. টেবিল এক্সপোর্ট করা (CSV)

পরিসংখ্যান (যেমন- প্রতিটি জেলার গড় বৃষ্টিপাত) একটি স্প্রেডশিটে সংরক্ষণ করতে:

=== "JavaScript"
    ```javascript
    // একটি অঞ্চলের জন্য গড় বৃষ্টিপাত গণনা করুন
    var stats = rainfall.reduceRegion({
      reducer: ee.Reducer.mean(),
      geometry: roi,
      scale: 5000
    });

    // পরিসংখ্যান সহ একটি ফিচার তৈরি করুন
    var feature = ee.Feature(null, stats);
    var collection = ee.FeatureCollection([feature]);
    
    // CSV-তে এক্সপোর্ট করুন
    Export.table.toDrive({
      collection: collection,
      description: 'Rainfall_Stats_2023',
      fileFormat: 'CSV'
    });
    ```

=== "Python"
    ```python
    # পরিসংখ্যান গণনা করুন
    stats = rainfall.reduceRegion(
      reducer=ee.Reducer.mean(),
      geometry=roi,
      scale=5000
    )

    # একটি ফিচার কালেকশন তৈরি করুন
    feature = ee.Feature(None, stats)
    collection = ee.FeatureCollection([feature])
    
    # geemap ব্যবহার করে এক্সপোর্ট করুন
    geemap.ee_export_vector_to_drive(
        collection, 
        description='Rainfall_Stats_2023', 
        fileFormat='CSV'
    )
    ```

## ৩. চার্ট তৈরি করা
