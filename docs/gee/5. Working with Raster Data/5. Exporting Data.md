# Exporting Your Data

Once you've analyzed an image or made a map, you probably want to save it!
You can export results to **Google Drive**, **Google Cloud Storage**, or as an **Earth Engine Asset**.

## 1. Exporting an Image (GeoTIFF)

To download a map (like an NDVI image) to your Google Drive:

=== "JavaScript"
    ```javascript
    // Select the image you want to export (e.g., NDVI)
    var imageToExport = easyNDVI;

    // Define the export parameters
    Export.image.toDrive({
      image: imageToExport,
      description: 'NDVI_Export_2023',
      scale: 30, // Resolution in meters (30m for Landsat)
      region: roi, // The area to export
      fileFormat: 'GeoTIFF'
    });
    ```
    *A new task will appear in the "Tasks" tab on the right. Click "Run" to start the download!*

=== "Python"
    ```python
    import geemap

    # Export using geemap (simplest way)
    geemap.ee_export_image_to_drive(
        easy_ndvi, 
        description='NDVI_Export_2023', 
        folder='GEE_Exports', 
        scale=30, 
        region=roi
    )
    ```

## 2. Exporting a Table (CSV)

To save statistics (like average rainfall per district) to a spreadsheet:

=== "JavaScript"
    ```javascript
    // Calculate mean rainfall for a region
    var stats = rainfall.reduceRegion({
      reducer: ee.Reducer.mean(),
      geometry: roi,
      scale: 5000
    });

    // Create a feature with the stats
    var feature = ee.Feature(null, stats);
    var collection = ee.FeatureCollection([feature]);
    
    // Export to CSV
    Export.table.toDrive({
      collection: collection,
      description: 'Rainfall_Stats_2023',
      fileFormat: 'CSV'
    });
    ```

=== "Python"
    ```python
    # Calculate stats
    stats = rainfall.reduceRegion(
      reducer=ee.Reducer.mean(),
      geometry=roi,
      scale=5000
    )

    # Create a feature collection
    feature = ee.Feature(None, stats)
    collection = ee.FeatureCollection([feature])
    
    # Export using geemap
    geemap.ee_export_vector_to_drive(
        collection, 
        description='Rainfall_Stats_2023', 
        fileFormat='CSV'
    )
    ```

## 3. Creating Charts

If you just want a quick graph, you can print charts directly in the Code Editor (JavaScript) or display them in Jupyter (Python) without exporting files.

- **JavaScript**: Use `ui.Chart` functions.
- **Python**: Use `geemap.chart` or `matplotlib`.
