# একক ব্যান্ড ইমেজ (Single Band Images) নিয়ে কাজ করা

একটি **একক ব্যান্ড ইমেজ (Single Band Image)** অনেকটা সাদা-কালো ছবির মতো যেখানে প্রতিটি পিক্সেল একটি নির্দিষ্ট মান ধারণ করে। রঙের (লাল/সবুজ/নীল) পরিবর্তে এতে থাকতে পারে:

- **উচ্চতা বা ইলিভেশন (Elevation)**: সমুদ্রপৃষ্ঠ থেকে উচ্চতা
- **তাপমাত্রা (Temperature)**: কতটা গরম বা ঠান্ডা
- **বৃষ্টিপাত (Rainfall)**: কতটুকু বৃষ্টি হয়েছে

## উদাহরণ ১: উচ্চতা সংক্রান্ত ডেটা (SRTM)

আমরা **SRTM** ডেটাসেট ব্যবহার করব, যা সমগ্র বিশ্বের ভূমির উচ্চতার তথ্য প্রদান করে।

=== "JavaScript"
    ```javascript
    // SRTM উচ্চতা বা ইলিভেশন ডেটা লোড করুন
    var srtm = ee.Image("CGIAR/SRTM90_V4");

    // কোনো একটি পাহাড়ি এলাকার (হিমালয়) ওপর ম্যাপটি কেন্দ্র করুন
    Map.setCenter(86.92, 27.98, 8);
    
    // ভিজ্যুয়ালাইজেশন প্যারামিটারসহ লেয়ারটি যুক্ত করুন
    // min: 0 মিটার (সমুদ্রপৃষ্ঠ), max: 6000 মিটার (উঁচু শৃঙ্গ)
    Map.addLayer(srtm, {min: 0, max: 6000}, 'Elevation');
    ```

=== "Python"
    ```python
    import ee
    import geemap

    # ইনিশালাইজ করুন
    ee.Initialize()
    
    # SRTM উচ্চতা বা ইলিভেশন ডেটা লোড করুন
    srtm = ee.Image("CGIAR/SRTM90_V4")
    
    # একটি ম্যাপ তৈরি করুন
    Map = geemap.Map()
    
    # কোনো একটি পাহাড়ি এলাকার (হিমালয়) ওপর ম্যাপটি কেন্দ্র করুন
    Map.setCenter(86.92, 27.98, 8)
    
    # ভিজ্যুয়ালাইজেশন প্যারামিটারসহ লেয়ারটি যুক্ত করুন
    Map.addLayer(srtm, {'min': 0, 'max': 6000}, 'Elevation')
    Map
    ```

**কী ঘটছে?**

- গাঢ় বা কালো পিক্সেল = কম উচ্চতা
- উজ্জ্বল বা সাদা পিক্সেল = বেশি উচ্চতা

## উদাহরণ ২: বৃষ্টিপাতের ডেটা (CHIRPS)

**CHIRPS** ডেটাসেট প্রতিদিনের বৃষ্টিপাত ট্র্যাক করে। চলুন এক বছরের মোট বৃষ্টিপাতের পরিমাণ দেখি।

=== "JavaScript"
    ```javascript
    // ২০২৩ সালের বৃষ্টিপাতের ডেটা লোড করুন
    var rainfall = ee.ImageCollection("UCSB-CHG/CHIRPS/DAILY")
      .filterDate('2023-01-01', '2023-12-31')
      .sum(); // বার্ষিক মোট বৃষ্টিপাত পেতে প্রতিদিনের বৃষ্টিপাত যোগ করুন

    // ভিজ্যুয়ালাইজেশন: নীল = বেশি বৃষ্টিপাত, সাদা = কম বৃষ্টিপাত
    var rainVis = {
      min: 0,
      max: 2000,
      palette: ['white', 'blue', 'darkblue']
    };
    
    Map.setCenter(78.9, 20.5, 5); // ভারত ও পার্শ্ববর্তী অঞ্চল
    Map.addLayer(rainfall, rainVis, 'Total Rainfall 2023');
    ```

=== "Python"
    ```python
    # ২০২৩ সালের বৃষ্টিপাতের ডেটা লোড করুন
    rainfall = ee.ImageCollection("UCSB-CHG/CHIRPS/DAILY") \
      .filterDate('2023-01-01', '2023-12-31') \
      .sum() # বার্ষিক মোট বৃষ্টিপাত পেতে প্রতিদিনের বৃষ্টিপাত যোগ করুন

    # ভিজ্যুয়ালাইজেশন: নীল = বেশি বৃষ্টিপাত, সাদা = কম বৃষ্টিপাত
    rain_vis = {
      'min': 0,
      'max': 2000,
      'palette': ['white', 'blue', 'darkblue']
    }
    
    Map.setCenter(78.9, 20.5, 5) # ভারত ও পার্শ্ববর্তী অঞ্চল
    Map.addLayer(rainfall, rain_vis, 'Total Rainfall 2023')
    Map
    ```
