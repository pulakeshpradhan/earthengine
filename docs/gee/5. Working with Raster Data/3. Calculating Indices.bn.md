# সূচক গণনা করা: যেমন NDVI (Calculating Indices)

স্যাটেলাইট ইমেজের মধ্যে অনেক লুকানো তথ্য থাকে। এর বিভিন্ন ব্যান্ড বা রঙের ওপর গাণিতিক হিসাব করে আমরা গাছপালার স্বাস্থ্য বা জলের উপস্থিতির মতো বিষয়গুলো উন্মোচন করতে পারি।

সবচেয়ে বিখ্যাত সূচক বা ইনডেক্স হলো **NDVI** (Normalized Difference Vegetation Index)।

## NDVI কী?

গাছপালা সালোকসংশ্লেষণের জন্য **লাল আলো (Red light)** শোষণ করে এবং **ইনফ্রারেড আলো (Infrared light)** প্রতিফলিত করে। মৃত গাছপালা বা মাটি উভয় আলোকেই সমানভাবে প্রতিফলিত করে।

**সূত্র (Formula):**
$$ NDVI = \frac{(NIR - Red)}{(NIR + Red)} $$

হিসাব নিয়ে চিন্তা করবেন না - আর্থ ইঞ্জিন আপনার হয়ে এই কাজটি করে দেয়!

## আর্থ ইঞ্জিন-এ NDVI গণনা করা

চলুন একটি ল্যান্ডস্যাট ৮ ইমেজের জন্য NDVI গণনা করি।

=== "JavaScript"
    ```javascript
    // একটি ল্যান্ডস্যাট ৮ ইমেজ লোড করুন
    var image = ee.Image('LANDSAT/LC08/C02/T1_L2/LC08_044034_20140318');

    // পদ্ধতি ১: সরাসরি সূত্র ব্যবহার করে (একটু কঠিন)
    var ndvi = image.expression(
      '(NIR - RED) / (NIR + RED)', {
        'NIR': image.select('SR_B5'),
        'RED': image.select('SR_B4')
      });
      
    // পদ্ধতি ২: বিল্ট-ইন ফাংশন ব্যবহার করে (সহজ!)
    var easyNDVI = image.normalizedDifference(['SR_B5', 'SR_B4']).rename('NDVI');
    
    // ভিজ্যুয়ালাইজেশন: নীল (জল) -> সাদা (খালি জমি) -> সবুজ (স্বাস্থ্যকর গাছপালা)
    var ndviVis = {min: -1, max: 1, palette: ['blue', 'white', 'green']};
    
    Map.centerObject(image, 9);
    Map.addLayer(easyNDVI, ndviVis, 'NDVI');
    ```

=== "Python"
    ```python
    # একটি ল্যান্ডস্যাট ৮ ইমেজ লোড করুন
    image = ee.Image('LANDSAT/LC08/C02/T1_L2/LC08_044034_20140318')

    # পদ্ধতি ২: বিল্ট-ইন ফাংশন ব্যবহার করে (সহজ!)
    # আমরা ৫ নম্বর (NIR) এবং ৪ নম্বর (Red) ব্যান্ড নির্বাচন করছি
    easy_ndvi = image.normalizedDifference(['SR_B5', 'SR_B4']).rename('NDVI')
    
    # ভিজ্যুয়ালাইজেশন: নীল (জল) -> সাদা (খালি জমি) -> সবুজ (স্বাস্থ্যকর গাছপালা)
    ndvi_vis = {'min': -1, 'max': 1, 'palette': ['blue', 'white', 'green']}
    
    Map = geemap.Map()
    Map.centerObject(image, 9)
    Map.addLayer(easy_ndvi, ndvi_vis, 'NDVI')
    Map
    ```

## ফলাফলের ব্যাখ্যা

- **মান +১ এর কাছাকাছি** (গাঢ় সবুজ): ঘন ও স্বাস্থ্যকর বন।
- **মান ০.২ - ০.৫ এর মধ্যে** (হালকা সবুজ): ঘাস বা পাতলা গাছপালা।
- **মান ০ এর কাছাকাছি** (সাদা/বাদামী): শহর এলাকা, খালি মাটি, পাথর।
- **ঋণাত্মক মান (Negative Values)** (নীল): জলাশয় বা নদী।

## অন্যান্য সাধারণ সূচকসমূহ

| ইনডেক্স | নাম | সূত্র | ব্যবহারের ক্ষেত্র |
| :--- | :--- | :--- | :--- |
| **NDWI** | Normalized Difference Water Index | (Green - NIR) / (Green + NIR) | জলাশয় শনাক্ত করা |
| **NBR** | Normalized Burn Ratio | (NIR - SWIR) / (NIR + SWIR) | অগ্নিকাণ্ডের পর পুড়ে যাওয়া এলাকা শনাক্ত করা |
| **EVI** | Enhanced Vegetation Index | জটিল সূত্র | ঘন বনে গাছপালা পর্যবেক্ষণের জন্য আরও নিখুঁত |

আপনি `.normalizedDifference()` অথবা `.expression()` ব্যবহার করে এগুলোর যেকোনোটি গণনা করতে পারেন!
