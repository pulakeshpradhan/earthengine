# Working with Single Band Images

A **Single Band Image** is like a grayscale photo where each pixel holds one value.
Instead of colors (Red/Green/Blue), it might hold:

- **Elevation** (Height above sea level)
- **Temperature** (How hot it is)
- **Rainfall** (How much it rained)

## Example 1: Elevation Data (SRTM)

We will use the **SRTM** dataset, which gives the height of the land for the entire world.

=== "JavaScript"
    ```javascript
    // Load the SRTM Elevation data
    var srtm = ee.Image("CGIAR/SRTM90_V4");

    // Center the map on a mountainous area (Himalayas)
    Map.setCenter(86.92, 27.98, 8);
    
    // Add the layer with visualization parameters
    // min: 0 meters (sea level), max: 6000 meters (high peaks)
    Map.addLayer(srtm, {min: 0, max: 6000}, 'Elevation');
    ```

=== "Python"
    ```python
    import ee
    import geemap

    # Initialize
    ee.Initialize()
    
    # Load the SRTM Elevation data
    srtm = ee.Image("CGIAR/SRTM90_V4")
    
    # Create a map
    Map = geemap.Map()
    
    # Center the map on a mountainous area (Himalayas)
    Map.setCenter(86.92, 27.98, 8)
    
    # Add the layer with visualization parameters
    Map.addLayer(srtm, {'min': 0, 'max': 6000}, 'Elevation')
    Map
    ```

**What is happening?**

- Darker pixels = Lower elevation
- Brighter pixels = Higher elevation

## Example 2: Rainfall Data (CHIRPS)

The **CHIRPS** dataset tracks rainfall daily. Let's look at total rainfall for a year.

=== "JavaScript"
    ```javascript
    // Load rainfall data for 2023
    var rainfall = ee.ImageCollection("UCSB-CHG/CHIRPS/DAILY")
      .filterDate('2023-01-01', '2023-12-31')
      .sum(); // Add up all daily rain to get yearly total

    // Visualization: Blue = High rainfall, White = Low rainfall
    var rainVis = {
      min: 0,
      max: 2000,
      palette: ['white', 'blue', 'darkblue']
    };
    
    Map.setCenter(78.9, 20.5, 5); // India
    Map.addLayer(rainfall, rainVis, 'Total Rainfall 2023');
    ```

=== "Python"
    ```python
    # Load rainfall data for 2023
    rainfall = ee.ImageCollection("UCSB-CHG/CHIRPS/DAILY") \
      .filterDate('2023-01-01', '2023-12-31') \
      .sum() # Add up all daily rain to get yearly total

    # Visualization: Blue = High rainfall, White = Low rainfall
    rain_vis = {
      'min': 0,
      'max': 2000,
      'palette': ['white', 'blue', 'darkblue']
    }
    
    Map.setCenter(78.9, 20.5, 5) # India
    Map.addLayer(rainfall, rain_vis, 'Total Rainfall 2023')
    Map
    ```
