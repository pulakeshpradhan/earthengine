# মাল্টি-ব্যান্ড ইমেজ (Multi-Band Images) নিয়ে কাজ করা

বেশিরভাগ স্যাটেলাইট ইমেজ হলো **মাল্টি-ব্যান্ড (Multi-Band)**, যার অর্থ হলো এগুলো বিভিন্ন ধরণের আলোকে আলাদাভাবে ধারণ করে।

এটিকে এমন একটি ক্যামেরার সাথে তুলনা করা যেতে পারে যা একসাথে ১০টি ছবি তোলে:

- **লাল** আলো (Band 4)
- **সবুজ** আলো (Band 3)
- **নীল** আলো (Band 2)
- **ইনফ্রারেড** আলো (এমন তথ্য যা আমাদের চোখ দেখতে পায় না!)

এই ব্যান্ডগুলোকে একত্রিত করে আমরা পৃথিবীকে বিভিন্নভাবে দেখতে পাই।

## ১. ট্রু কালার (True Color - যা আমরা দেখি)

পৃথিবীকে আমাদের চোখ যেভাবে দেখে সেভাবে দেখতে আমরা লাল, সবুজ এবং নীল ব্যান্ডগুলোকে একত্রিত করি।

=== "JavaScript"
    ```javascript
    // একটি ল্যান্ডস্যাট ৮ ইমেজ লোড করুন
    var image = ee.Image('LANDSAT/LC08/C02/T1_L2/LC08_044034_20140318');

    // ব্যান্ড ৪ (লাল), ৩ (সবুজ), ২ (নীল) একত্রিত করুন
    var trueColor = {
      bands: ['SR_B4', 'SR_B3', 'SR_B2'],
      min: 7000,
      max: 12000
    };
    
    Map.centerObject(image, 9);
    Map.addLayer(image, trueColor, 'True Color');
    ```

=== "Python"
    ```python
    # একটি ল্যান্ডস্যাট ৮ ইমেজ লোড করুন
    image = ee.Image('LANDSAT/LC08/C02/T1_L2/LC08_044034_20140318')

    # ব্যান্ড ৪ (লাল), ৩ (সবুজ), ২ (নীল) একত্রিত করুন
    true_color = {
        'bands': ['SR_B4', 'SR_B3', 'SR_B2'],
        'min': 7000,
        'max': 12000
    }
    
    Map = geemap.Map()
    Map.centerObject(image, 9)
    Map.addLayer(image, true_color, 'True Color')
    Map
    ```

## ২. ফলস কালার (False Color - গাছপালার সুস্থতা)

গাছপালা ব্যাপকভাবে **ইনফ্রারেড (Infrared)** আলো প্রতিফলিত করে, যা মানুষ দেখতে পায় না। ব্যান্ডগুলোকে অদলবদল করে আমরা গাছপালাকে **উজ্জ্বল লাল (Bright Red)** রঙে ফুটিয়ে তুলতে পারি।

- **নিয়ার ইনফ্রারেড** (Band 5) -> লাল হিসেবে দেখাবে
- **লাল** (Band 4) -> সবুজ হিসেবে দেখাবে
- **সবুজ** (Band 3) -> নীল হিসেবে দেখাবে

=== "JavaScript"
    ```javascript
    // ব্যান্ড ৫ (NIR), ৪ (লাল), ৩ (সবুজ) একত্রিত করুন
    var falseColor = {
      bands: ['SR_B5', 'SR_B4', 'SR_B3'],
      min: 7000,
      max: 12000
    };

    Map.addLayer(image, falseColor, 'False Color (Vegetation)');
    ```

=== "Python"
    ```python
    # ব্যান্ড ৫ (NIR), ৪ (লাল), ৩ (সবুজ) একত্রিত করুন
    false_color = {
        'bands': ['SR_B5', 'SR_B4', 'SR_B3'],
        'min': 7000,
        'max': 12000
    }

    Map.addLayer(image, false_color, 'False Color (Vegetation)')
    ```

**এর মানে কী?**

- **উজ্জ্বল লাল** = স্বাস্থ্যকর, ঘন গাছপালা
- **গাঢ় লাল** = পাতলা বা কম গাছপালা
- **সায়ান/নীল** = শহর, খালি মাটি বা জলভাগ

## কেন ইনফ্রারেড ব্যবহার করবেন?

ইনফ্রারেড **আসল গাছপালা** এবং সবুজ রঙের বস্তুর (যেমন কৃত্রিম ঘাস বা সবুজ রঙ) মধ্যে পার্থক্য করতে সাহায্য করে। ইনফ্রারেডে কেবল আসল গাছপালাই উজ্জ্বল দেখায়!
