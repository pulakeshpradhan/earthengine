# টাইম সিরিজ বিশ্লেষণ (সময়ের সাথে পরিবর্তন)

স্যাটেলাইট ডেটা শক্তিশালী কারণ আমাদের কাছে কয়েক দশক আগের ছবি আছে। অনেকগুলো ছবির "স্তূপ" বিশ্লেষণ করে আমরা দেখতে পাই কোনো একটি জায়গা কীভাবে বদলেছে।

## `map()` ফাংশন

অনেকগুলো ছবিকে একসাথে প্রসেস করতে (যেমন- একটি বছরের প্রতিটি ছবির জন্য NDVI গণনা করতে) আমরা `.map()` ব্যবহার করি।
এটিকে একটি **অ্যাসেম্বলি লাইন** এর মতো ভাবুন: আপনি নির্দেশনা তৈরি করে দেন, এবং আর্থ ইঞ্জিন সেই নির্দেশনাগুলো কালেকশনের প্রতিটি ছবিতে স্বয়ংক্রিয়ভাবে প্রয়োগ করে।

## উদাহরণ: এক বছরে গাছপালার স্বাস্থ্যের চার্ট তৈরি করা

চলুন ২০২৩ সালের প্রতিটি ল্যান্ডস্যাট ৮ ছবির জন্য NDVI গণনা করি এবং একটি চার্টে তা প্রদর্শন করি।

=== "JavaScript"
    ```javascript
    // একটি নির্দিষ্ট অবস্থান ডিফাইন করুন (যেমন- একটি বন বা কৃষিক্ষেত্র)
    var roi = ee.Geometry.Point([-122.4, 37.8]); // সান ফ্রান্সিসকো

    // ১ বছরের জন্য ল্যান্ডস্যাট ৮ লোড করুন
    var collection = ee.ImageCollection('LANDSAT/LC08/C02/T1_L2')
      .filterDate('2023-01-01', '2023-12-31')
      .filterBounds(roi);
      
    // NDVI গণনা করার জন্য একটি ফাংশন তৈরি করুন
    var addNDVI = function(image) {
      var ndvi = image.normalizedDifference(['SR_B5', 'SR_B4']).rename('NDVI');
      return image.addBands(ndvi); // একটি নতুন ব্যান্ড হিসেবে NDVI যোগ করুন
    };
    
    // কালেকশনের প্রতিটি ইমেজে ফাংশনটি প্রয়োগ করুন!
    var withNDVI = collection.map(addNDVI);
    
    // একটি চার্ট তৈরি করুন
    print(ui.Chart.image.series({
      imageCollection: withNDVI.select('NDVI'),
      region: roi,
      reducer: ee.Reducer.mean(),
      scale: 30
    }).setOptions({
      title: 'NDVI Over Time (2023)',
      vAxis: {title: 'NDVI'},
      hAxis: {title: 'Date'}
    }));
    ```

=== "Python"
    ```python
    # একটি নির্দিষ্ট অবস্থান ডিফাইন করুন
    roi = ee.Geometry.Point([-122.4, 37.8]) # সান ফ্রান্সিসকো

    # ১ বছরের জন্য ল্যান্ডস্যাট ৮ লোড করুন
    collection = ee.ImageCollection('LANDSAT/LC08/C02/T1_L2') \
      .filterDate('2023-01-01', '2023-12-31') \
      .filterBounds(roi)
      
    # NDVI গণনা করার জন্য একটি ফাংশন তৈরি করুন
    def add_ndvi(image):
      ndvi = image.normalizedDifference(['SR_B5', 'SR_B4']).rename('NDVI')
      return image.addBands(ndvi)
    
    # প্রতিটি ইমেজে ফাংশনটি প্রয়োগ করুন
    with_ndvi = collection.map(add_ndvi)
    
    # geemap ব্যবহার করে চার্ট তৈরি করুন
    import geemap.chart as chart
    
    chart.feature_byFeature(
        with_ndvi,
        roi,
        'NDVI',
        title='NDVI Over Time (2023)',
        x_label='Date',
        y_label='NDVI'
    )
    # বিকল্পভাবে, আপনি পাইথনে matplotlib-ও ব্যবহার করতে পারেন!
    ```
