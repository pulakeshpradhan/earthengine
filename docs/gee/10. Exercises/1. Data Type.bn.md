# গুগল আর্থ ইঞ্জিন অনুশীলনী

---

## মডিউল ১: আর্থ ইঞ্জিন মৌলিক বিষয়

### ০১. হ্যালো ওয়ার্ল্ড

- বিশ্বের পাঁচটি বৃহত্তম শহরের তালিকা করুন: টোকিও, দিল্লি, সাংহাই, মেক্সিকো সিটি, সাও পাওলো।
- এই শহরের নাম দিয়ে `largeCities` নামে একটি লিস্ট তৈরি করুন।
- কনসোলে লিস্ট প্রিন্ট করুন।

### ০২. ইমেজ কালেকশনে কাজ করা

- [আর্থ ইঞ্জিন ক্যাটালগ](https://developers.google.com/earth-engine/datasets) থেকে *Sentinel-2 Level-1C* ডেটাসেট খুঁজুন।
- ডেটাসেট লোড করার কোড স্নিপেট কপি-পেস্ট করুন।
- আপনার শহরের জন্য ইমেজ প্রদর্শনে কোড পরিবর্তন করুন।

### ০৩. ইমেজ কালেকশন ফিল্টারিং

- নির্দিষ্ট শর্তে ইমেজ কালেকশনে ফিল্টার প্রয়োগ করুন:
  - `ee.Filter.eq()`, `ee.Filter.lt()` ইত্যাদি ব্যবহার করে।
  - `ee.Filter.date()` দিয়ে তারিখ পরিসর।
  - `ee.Filter.bounds()` দিয়ে অবস্থান।

### ০৪. ইমেজ কালেকশন থেকে মোজাইক ও কম্পোজিট তৈরি

- নির্দিষ্ট সময়ের জন্য মিডিয়ান কম্পোজিট ইমেজ তৈরি করুন।

### ০৫. ফিচার কালেকশনে কাজ করা

- নির্দিষ্ট অঞ্চল নির্বাচনে ফিচার কালেকশন ফিল্টার করুন।

### ০৬. ডেটা ইম্পোর্ট করা

- একটি ডেটাসেট ইম্পোর্ট করুন এবং নির্দিষ্ট ফিচার নির্বাচনে ফিল্টার প্রয়োগ করুন।

### ০৭. ইমেজ ক্লিপিং

- একটি নির্দিষ্ট ফিচারের জ্যামিতিতে ইমেজ ক্লিপ করুন।

### ০৮. ডেটা এক্সপোর্ট করা

- প্রক্রিয়াকৃত ইমেজ বা ফিচার কালেকশন অ্যাসেট হিসেবে এক্সপোর্ট করুন।

---

## মডিউল ২: আর্থ ইঞ্জিন মধ্যবর্তী স্তর

### ০১. আর্থ ইঞ্জিন অবজেক্ট

- ডেটা প্রক্রিয়াকরণে আর্থ ইঞ্জিন অবজেক্ট ম্যানিপুলেট করুন।

### ০২. সূচক গণনা

- NDVI বা EVI এর মতো স্পেকট্রাল সূচক গণনা করুন।

### ০৩. ইমেজ কালেকশনে গণনা

- রিডিউসার ব্যবহার করে ইমেজ কালেকশনে গণনা সম্পাদন করুন।

### ০৪. ক্লাউড মাস্কিং

- ইমেজ মান উন্নত করতে ক্লাউড মাস্কিং কৌশল প্রয়োগ করুন।

### ০৫. রিডিউসার

- ইমেজ কালেকশন থেকে ডেটা সারসংক্ষেপ করতে রিডিউসার ব্যবহার করুন।

### ০৬. টাইম-সিরিজ চার্ট

- সময়ের সাথে পরিবর্তন ভিজ্যুয়ালাইজ করতে টাইম-সিরিজ চার্ট তৈরি করুন।

---

## মডিউল ৩: তত্ত্বাবধিত শ্রেণীবিভাগ

### ০১. মৌলিক তত্ত্বাবধিত শ্রেণীবিভাগ

- প্রশিক্ষণ ডেটা সহ তত্ত্বাবধিত শ্রেণীবিভাগ সম্পাদন করুন।

### ০২. নির্ভুলতা মূল্যায়ন

- শ্রেণীবিভক্ত ইমেজের জন্য নির্ভুলতা মেট্রিক্স গণনা করুন।

### ০৩. শ্রেণীবিভাগ উন্নতি

- স্পেকট্রাল সূচক যোগ করে শ্রেণীবিভাগ ফলাফল উন্নত করুন।

### ০৪. শ্রেণীবিভাগ ফলাফল এক্সপোর্ট

- শ্রেণীবিভক্ত ইমেজ অ্যাসেট হিসেবে এক্সপোর্ট করুন।

### ০৫. ক্ষেত্রফল গণনা

- বিভিন্ন ভূমি আচ্ছাদন শ্রেণীর ক্ষেত্রফল গণনা করুন।

---

## মডিউল ৪: পরিবর্তন সনাক্তকরণ

### ০১. স্পেকট্রাল সূচক পরিবর্তন

- সময়ের সাথে স্পেকট্রাল সূচকের পরিবর্তন বিশ্লেষণ করুন।

### ০২. স্পেকট্রাল দূরত্ব পরিবর্তন

- পরিবর্তন সনাক্তে স্পেকট্রাল দূরত্ব পদ্ধতি প্রয়োগ করুন।

### ০৩. পরিবর্তনের সরাসরি শ্রেণীবিভাগ

- মেশিন লার্নিং কৌশল ব্যবহার করে সরাসরি পরিবর্তন শ্রেণীবিভাগ করুন।

### ০৪. পোস্ট-ক্লাসিফিকেশন তুলনা

- প্রি- ও পোস্ট-ক্লাসিফিকেশন ফলাফল তুলনা করুন।

## মডিউল ৫: আর্থ ইঞ্জিন অ্যাপস

### ০১. ক্লায়েন্ট বনাম সার্ভার

- ক্লায়েন্ট-সাইড ও সার্ভার-সাইড প্রক্রিয়াকরণের পার্থক্য বুঝুন।

### ০২. UI এলিমেন্ট ব্যবহার

- ড্রপডাউন ও বাটনের মতো UI এলিমেন্ট তৈরি করুন।

### ০৩. অ্যাপ তৈরি ও প্রকাশ

- একটি ইন্টারেক্টিভ আর্থ ইঞ্জিন অ্যাপ তৈরি ও প্রকাশ করুন।

### ০৪. স্প্লিট প্যানেল অ্যাপ তৈরি

- একাধিক ভিউ সহ স্প্লিট-প্যানেল অ্যাপ ডিজাইন করুন।

## মডিউল ৬: গুগল আর্থ ইঞ্জিন পাইথন API

### ০১. পাইথন API সিনট্যাক্স

- আর্থ ইঞ্জিন কোড পাইথন API সিনট্যাক্সে রূপান্তর করুন।

### ০২. স্বয়ংক্রিয় কোড রূপান্তর

- *geemap* ব্যবহার করে স্বয়ংক্রিয়ভাবে আর্থ ইঞ্জিন কোড পাইথনে রূপান্তর করুন।

### ০৩. ব্যাচ এক্সপোর্ট

- একাধিক ইমেজের জন্য ব্যাচ এক্সপোর্ট টাস্ক শুরু করুন।

### ০৪. XArray-এর সাথে আর্থ ইঞ্জিন ব্যবহার

- উন্নত ডেটা বিশ্লেষণের জন্য XArray-এর সাথে আর্থ ইঞ্জিন সংহত করুন।

### ০৫. স্বয়ংক্রিয় ডাউনলোড

- পাইথন API ব্যবহার করে ডেটা ডাউনলোড স্বয়ংক্রিয় করুন।

### ০৬. স্বয়ংক্রিয় এক্সপোর্ট

- পাইথন API ও সার্ভিস অ্যাকাউন্ট দিয়ে এক্সপোর্ট স্বয়ংক্রিয় করুন।

### ০৭. QGIS প্লাগইন ব্যবহার

- আর্থ ইঞ্জিন ডেটা ভিজ্যুয়ালাইজ করতে QGIS প্লাগইন ব্যবহার করুন।
