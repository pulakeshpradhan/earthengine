# जावास्क्रिप्ट बनाम पायथन: क्या अंतर है? (JavaScript vs Python: What's the Difference?)

गूगल अर्थ इंजन दो भाषाओं के साथ काम करता है: **जावास्क्रिप्ट (JavaScript)** और **पायथन (Python)**।
GEE में ये दोनों भाषाएँ एक ही काम करती हैं, लेकिन उनके लिखने का तरीका थोड़ा अलग है।

## त्वरित तुलना (Quick Comparison)

| विशेषता | जावास्क्रिप्ट (कोड एडिटर) | पायथन (जुपीटर/कोलाब) |
| :--- | :--- | :--- |
| **वेरिएबल्स (Variables)** | `var image = ...` | `image = ...` (कोई `var` नहीं) |
| **फ़ंक्शंस (Functions)** | `function myFunc(x) { ... }` | `def my_func(x):` |
| **कमेंट (Comments)** | `// This is a comment` | `# This is a comment` |
| **सही/गलत (True/False)** | `true`, `false` (छोटे अक्षरों में) | `True`, `False` (बड़े अक्षरों से शुरू) |
| **शून्य (Null)** | `null` | `None` |
| **डिक्शनरी** | `{key: 'value'}` | `{'key': 'value'}` (कीज़ पर उद्धरण चिह्न ज़रूरी) |
| **इंस्टालेशन?** | नहीं (ब्राउज़र में चलता है) | हाँ (सेटअप की आवश्यकता है) |

---

## कोड उदाहरण (Code Examples)

यहाँ दोनों भाषाओं में सामान्य कार्यों को लिखने का तरीका दिया गया है।

### 1. छवि लोड करना (Loading an Image)

=== "JavaScript"
    ```javascript
    // लैंडसैट 8 लोड करें
    var image = ee.Image('LANDSAT/LC08/C02/T1_L2/LC08_044034_20140318');

    print('Image loaded:', image);
    ```

=== "Python"
    ```python
    # लैंडसैट 8 लोड करें
    image = ee.Image('LANDSAT/LC08/C02/T1_L2/LC08_044034_20140318')

    print('Image loaded:', image.getInfo()) # ध्यान दें: प्रिंट करने के लिए .getInfo() ज़रूरी है
    ```

### 2. विज़ुअलाइज़ेशन पैरामीटर्स को परिभाषित करना

यह सबसे बड़ा अंतर है! पायथन में डिक्शनरी की 'कीज़' (keys) के चारों ओर **उद्धरण चिह्नों (' ')** की आवश्यकता होती है।

=== "JavaScript"
    ```javascript
    var visParams = {
      bands: ['SR_B4', 'SR_B3', 'SR_B2'],
      min: 7000,
      max: 12000,
      palette: ['blue', 'green', 'red']
    };

    Map.addLayer(image, visParams, 'My Image');
    ```

=== "Python"
    ```python
    vis_params = {
        'bands': ['SR_B4', 'SR_B3', 'SR_B2'],
        'min': 7000,
        'max': 12000,
        'palette': ['blue', 'green', 'red']
    }

    Map.addLayer(image, vis_params, 'My Image')
    ```

## मुझे किसका उपयोग करना चाहिए?

- **जावास्क्रिप्ट चुनें** यदि: आप एक शुरुआती (beginner) हैं। यह ब्राउज़र में तुरंत चलता है और सीखना सबसे आसान है।
- **पायथन चुनें** यदि: आप एक डेटा साइंटिस्ट हैं, अन्य लाइब्रेरीज़ (जैसे pandas/matplotlib) का उपयोग करना चाहते हैं, या अपने कार्यप्रवाह (workflow) को स्वचालित करना चाहते हैं।
