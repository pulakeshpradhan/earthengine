# জাভাস্ক্রিপ্ট বনাম পাইথন: পার্থক্য কী? (JavaScript vs Python: What's the Difference?)

গুগল আর্থ ইঞ্জিন দুটি প্রোগ্রামিং ভাষা নিয়ে কাজ করে: **জাভাস্ক্রিপ্ট (JavaScript)** এবং **পাইথন (Python)**।
GEE-তে উভয় ভাষাই একই কাজ করে, তবে তাদের লেখার ধরণ কিছুটা আলাদা।

## দ্রুত তুলনা (Quick Comparison)

| বৈশিষ্ট্য | জাভাস্ক্রিপ্ট (কোড এডিটর) | পাইথন (জুপিটার/কোলাব) |
| :--- | :--- | :--- |
| **ভ্যারিয়েবল বা চলক** | `var image = ...` | `image = ...` (কোনো `var` নেই) |
| **ফাংশন (Functions)** | `function myFunc(x) { ... }` | `def my_func(x):` |
| **কমেন্ট (Comments)** | `// This is a comment` | `# This is a comment` |
| **সত্য/মিথ্যা (True/False)** | `true`, `false` (ছোট হাতের) | `True`, `False` (বড় হাতের শুরুর অক্ষর) |
| **শূন্য বা নাল (Null)** | `null` | `None` |
| **ডিকশনারি** | `{key: 'value'}` | `{'key': 'value'}` (কী-তে উদ্ধৃতি প্রয়োজন) |
| **ইনস্টলেশন?** | না (ব্রাউজারে চলে) | হ্যাঁ (সেটআপ প্রয়োজন) |

## কোড উদাহরণ (Code Examples)

উভয় ভাষায় সাধারণ কাজগুলো কীভাবে লিখতে হয় তা নিচে দেওয়া হলো।

### ১. ইমেজ লোড করা

=== "JavaScript"
    ```javascript
    // ল্যান্ডস্যাট ৮ লোড করুন
    var image = ee.Image('LANDSAT/LC08/C02/T1_L2/LC08_044034_20140318');

    print('Image loaded:', image);
    ```

=== "Python"
    ```python
    # ল্যান্ডস্যাট ৮ লোড করুন
    image = ee.Image('LANDSAT/LC08/C02/T1_L2/LC08_044034_20140318')

    print('Image loaded:', image.getInfo()) # দ্রষ্টব্য: প্রিন্ট করার জন্য .getInfo() প্রয়োজন
    ```

### ২. ভিজ্যুয়ালাইজেশন প্যারামিটার নির্ধারণ

এটিই বড় পার্থক্য! পাইথনে ডিকশনারির কী-গুলোর (keys) দুপাশে **উদ্ধৃতি চিহ্ন (' ')** দিতে হয়।

=== "JavaScript"
    ```javascript
    var visParams = {
      bands: ['SR_B4', 'SR_B3', 'SR_B2'],
      min: 7000,
      max: 12000,
      palette: ['blue', 'green', 'red']
    };

    Map.addLayer(image, visParams, 'My Image');
    ```

=== "Python"
    ```python
    vis_params = {
        'bands': ['SR_B4', 'SR_B3', 'SR_B2'],
        'min': 7000,
        'max': 12000,
        'palette': ['blue', 'green', 'red']
    }

    Map.addLayer(image, vis_params, 'My Image')
    ```

## আমি কোনটি ব্যবহার করব?

- **জাভাস্ক্রিপ্ট বেছে নিন** যদি: আপনি একজন শিক্ষানবিস হন। এটি ব্রাউজারে তাৎক্ষণিকভাবে কাজ করে এবং শেখা সহজ।
- **পাইথন বেছে নিন** যদি: আপনি একজন ডেটা সায়েন্টিস্ট হন, অন্যান্য লাইব্রেরি (যেমন pandas/matplotlib) ব্যবহার করতে চান, অথবা কাজের প্রক্রিয়া স্বয়ংক্রিয় (automate) করতে চান।
