## সূচীপত্র

১. **মৌলিক প্রদর্শন** — `Map.addLayer()`, `Map.setCenter()`
২. **ডেটাসেট লোড** — `ee.Image()`, `ee.ImageCollection()`, `ee.FeatureCollection()`
৩. **ফিল্টারিং** — তারিখ, সীমানা, মেটাডেটা ফিল্টার
৪. **ব্যান্ড অপারেশন** — নির্বাচন, সূচক গণনা, গাণিতিক অপারেশন
৫. **জ্যামিতি** — পয়েন্ট, পলিগন, আয়তক্ষেত্র, বাফার
৬. **ডেটা রিডাকশন** — রিডিউসার, `reduceRegion()`, `reduceRegions()`
৭. **এক্সপোর্ট** — ইমেজ, টেবিল, ভিডিও এক্সপোর্ট
৮. **বিবিধ** — `aside()`, `evaluate()`, `getInfo()`

## মূল ধারণাসমূহ

### ক্লায়েন্ট বনাম সার্ভার

- **ক্লায়েন্ট-সাইড**: জাভাস্ক্রিপ্ট ভ্যারিয়েবল (var, string, number)
- **সার্ভার-সাইড**: আর্থ ইঞ্জিন অবজেক্ট (ee.Number, ee.String, ee.Image)
- সর্বদা সার্ভার-সাইড অপারেশন ব্যবহার করুন

### সাধারণ প্যাটার্ন

```javascript
// লোড > ফিল্টার > ম্যাপ > রিডিউস > ভিজ্যুয়ালাইজ
var result = ee.ImageCollection('ID')
  .filterDate(start, end)
  .filterBounds(roi)
  .map(function(img) { return img.normalizedDifference(['NIR', 'Red']); })
  .median();
Map.addLayer(result, visParams, 'ফলাফল');
```

### ভিজ্যুয়ালাইজেশন প্যারামিটার

```javascript
var visParams = {
  bands: ['B4', 'B3', 'B2'],  // RGB ব্যান্ড
  min: 0,                       // ন্যূনতম মান
  max: 3000,                    // সর্বোচ্চ মান
  palette: ['blue', 'green', 'red']  // রঙ প্যালেট
};
```
