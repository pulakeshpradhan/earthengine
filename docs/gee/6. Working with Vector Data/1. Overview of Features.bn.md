# ফিচার (পয়েন্ট, লাইন, পলিগন)

আর্থ ইঞ্জিন-এ একটি একক আকৃতিকে বলা হয় **ফিচার (Feature)**।
এটি হতে পারে একটি বিন্দু বা পয়েন্ট (যেমন- একটি শহর), একটি রেখা (যেমন- একটি রাস্তা), অথবা একটি বহুভুজ বা পলিগন (যেমন- একটি কৃষিক্ষেত্র)।

## ফিচার তৈরি করা

### ১. পয়েন্ট (Point)

=== "JavaScript"
    ```javascript
    // [দ্রাঘিমাংশ, অক্ষাংশ] এ একটি বিন্দু
    var point = ee.Geometry.Point([-122.2, 37.7]);

    Map.addLayer(point, {color: 'red'}, 'My Point');
    ```

=== "Python"
    ```python
    # [দ্রাঘিমাংশ, অক্ষাংশ] এ একটি বিন্দু
    point = ee.Geometry.Point([-122.2, 37.7])

    Map.addLayer(point, {'color': 'red'}, 'My Point')
    ```

### ২. লাইনস্ট্রিং (LineString)

একটি রেখা একাধিক বিন্দুকে সংযুক্ত করে।

=== "JavaScript"
    ```javascript
    var river = ee.Geometry.LineString([
      [-122.0, 37.0],
      [-122.1, 37.1],
      [-122.2, 37.2]
    ]);

    Map.addLayer(river, {color: 'blue'}, 'My River');
    ```

=== "Python"
    ```python
    river = ee.Geometry.LineString([
      [-122.0, 37.0],
      [-122.1, 37.1],
      [-122.2, 37.2]
    ])

    Map.addLayer(river, {'color': 'blue'}, 'My River')
    ```

### ৩. পলিগন (Polygon)

একটি পলিগন হলো একটি বদ্ধ আকৃতি (যেমন- আয়তক্ষেত্র বা যেকোনো কাস্টম আকৃতি)।

=== "JavaScript"
    ```javascript
    // [ন্যূনতম দ্রাঘিমা, ন্যূনতম অক্ষাংশ, সর্বোচ্চ দ্রাঘিমা, সর্বোচ্চ অক্ষাংশ] দিয়ে একটি আয়তক্ষেত্র
    var farm = ee.Geometry.Rectangle([-122.5, 37.5, -122.4, 37.6]);

    Map.addLayer(farm, {color: 'green'}, 'My Farm');
    ```

=== "Python"
    ```python
    # [ন্যূনতম দ্রাঘিমা, ন্যূনতম অক্ষাংশ, সর্বোচ্চ দ্রাঘিমা, সর্বোচ্চ অক্ষাংশ] দিয়ে একটি আয়তক্ষেত্র
    farm = ee.Geometry.Rectangle([-122.5, 37.5, -122.4, 37.6])

    Map.addLayer(farm, {'color': 'green'}, 'My Farm')
    ```
