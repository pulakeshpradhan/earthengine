# Feature Collections (Boundaries)

A **Feature Collection** is a group of features, like a bag full of shapes.
Most commonly, this is a dataset of country borders, states, or protected areas.

## 1. Loading Global Boundaries

Let's load all countries in the world.

=== "JavaScript"
    ```javascript
    // Load the dataset
    var countries = ee.FeatureCollection('USDOS/LSIB_SIMPLE/2017');

    // Display all countries (might be slow!)
    Map.addLayer(countries, {color: 'black'}, 'World Borders');
    ```

=== "Python"
    ```python
    # Load the dataset
    countries = ee.FeatureCollection('USDOS/LSIB_SIMPLE/2017')

    Map.addLayer(countries, {'color': 'black'}, 'World Borders')
    ```

## 2. Filtering a Specific Country

Usually, you only want one country. Use `.filter()` to find it.

=== "JavaScript"
    ```javascript
    // Find "France" by filtering the 'country_na' property
    var france = countries.filter(ee.Filter.eq('country_na', 'France'));

    // Center map and show only France
    Map.centerObject(france, 6);
    Map.addLayer(france, {color: 'blue'}, 'France');
    ```

=== "Python"
    ```python
    # Find "France"
    france = countries.filter(ee.Filter.eq('country_na', 'France'))

    Map.centerObject(france, 6)
    Map.addLayer(france, {'color': 'blue'}, 'France')
    ```

## 3. Clipping Images

The most useful thing you can do with a boundary is **cut (clip)** an image to fit inside it.

=== "JavaScript"
    ```javascript
    // Load elevation data
    var srtm = ee.Image("CGIAR/SRTM90_V4");

    // Clip the image to France's shape
    var franceElevation = srtm.clip(france);
    
    Map.addLayer(franceElevation, {min: 0, max: 2000}, 'Elevation (Clipped)');
    ```

=== "Python"
    ```python
    # Load elevation data
    srtm = ee.Image("CGIAR/SRTM90_V4")

    # Clip the image to France's shape
    france_elevation = srtm.clip(france)
    
    Map.addLayer(france_elevation, {'min': 0, 'max': 2000}, 'Elevation (Clipped)')
    ```
