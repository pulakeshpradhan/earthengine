### **ফিচার বনাম ফিচার কালেকশন**

* **ফিচার (Feature)**: সংশ্লিষ্ট বৈশিষ্ট্য (প্রপার্টি) সহ একটি একক ভৌগোলিক বস্তু। এটি একটি জ্যামিতি (যেমন- পয়েন্ট, লাইন, পলিগন) এবং মেটাডেটা নিয়ে গঠিত।
* **ফিচার কালেকশন (FeatureCollection)**: একাধিক ফিচারের সংকলন। এটি একাধিক ভৌগোলিক বস্তু ধারণকারী ডেটাসেট সংরক্ষণ ও প্রক্রিয়াকরণে ব্যবহৃত হয়।

---

### **FeatureCollection ইম্পোর্ট করা**

```javascript
var table = ee.FeatureCollection("projects/pulakesh/assets/cmc");
var city = ee.FeatureCollection("projects/pulakesh/assets/bmc");
Map.addLayer(table);
```

* `ee.FeatureCollection`: ইম্পোর্ট করা স্থানিক ডেটা প্রতিনিধিত্ব করে।
* `Map.addLayer`: ভিজ্যুয়ালাইজেশনের জন্য ডেটা মানচিত্রে যোগ করে।

---

### **১. ক্ষেত্রফল গণনা**

```javascript
print(table.geometry().area().divide(1e6));
```

* **উদ্দেশ্য**: সম্পূর্ণ FeatureCollection-এর ক্ষেত্রফল বর্গ কিলোমিটারে গণনা করে।

```javascript
var area = table.map(function(feature) {
  return feature.set('area_km2', feature.geometry().area().divide(1e6));
});
```

* প্রতিটি ফিচারে `area_km2` প্রপার্টি যোগ করে।

### **৩. বাফার (Buffer)**
