# ফিচার কালেকশন (সীমানাসমূহ)

একটি **ফিচার কালেকশন (Feature Collection)** হলো অনেকগুলো ফিচারের একটি গ্রুপ, অনেকটা আকৃতি ভর্তি একটি ব্যাগের মতো।
সাধারণত এটি দেশের সীমানা, রাজ্য বা সংরক্ষিত এলাকার একটি ডেটাসেট হয়।

## ১. বিশ্বব্যাপী সীমানা লোড করা

চলুন বিশ্বের সমস্ত দেশ লোড করি।

=== "JavaScript"
    ```javascript
    // ডেটাসেট লোড করুন
    var countries = ee.FeatureCollection('USDOS/LSIB_SIMPLE/2017');

    // সমস্ত দেশ দেখান (ধীর হতে পারে!)
    Map.addLayer(countries, {color: 'black'}, 'World Borders');
    ```

=== "Python"
    ```python
    # ডেটাসেট লোড করুন
    countries = ee.FeatureCollection('USDOS/LSIB_SIMPLE/2017')

    Map.addLayer(countries, {'color': 'black'}, 'World Borders')
    ```

## ২. একটি নির্দিষ্ট দেশ ফিল্টার করা

সাধারণত আপনি শুধু একটি দেশ চান। সেটি খুঁজতে `.filter()` ব্যবহার করুন।

=== "JavaScript"
    ```javascript
    // 'country_na' প্রোপার্টি ফিল্টার করে "France" খুঁজুন
    var france = countries.filter(ee.Filter.eq('country_na', 'France'));

    // ম্যাপকে ফ্রান্সের ওপর কেন্দ্র করুন এবং শুধু ফ্রান্স দেখান
    Map.centerObject(france, 6);
    Map.addLayer(france, {color: 'blue'}, 'France');
    ```

=== "Python"
    ```python
    # "France" খুঁজুন
    france = countries.filter(ee.Filter.eq('country_na', 'France'))

    Map.centerObject(france, 6)
    Map.addLayer(france, {'color': 'blue'}, 'France')
    ```

## ৩. ইমেজ ক্লিপিং (Clipping Images)

সীমানা দিয়ে সবচেয়ে কার্যকরী কাজটি হলো একটি ছবিকে সেই সীমানার ভেতরে কেটে ফেলা বা **ক্লিপ (clip)** করা।

=== "JavaScript"
    ```javascript
    // উচ্চতার ডেটা লোড করুন
    var srtm = ee.Image("CGIAR/SRTM90_V4");

    // ফ্রান্সের আকৃতি অনুযায়ী ইমেজটি ক্লিপ করুন
    var franceElevation = srtm.clip(france);
    
    Map.addLayer(franceElevation, {min: 0, max: 2000}, 'Elevation (Clipped)');
    ```

=== "Python"
    ```python
    # উচ্চতার ডেটা লোড করুন
    srtm = ee.Image("CGIAR/SRTM90_V4")

    # ফ্রান্সের আকৃতি অনুযায়ী ইমেজটি ক্লিপ করুন
    france_elevation = srtm.clip(france)
    
    Map.addLayer(france_elevation, {'min': 0, 'max': 2000}, 'Elevation (Clipped)')
    ```
