# ভেক্টর ডেটা এক্সপোর্ট করা (Exporting Vectors)

আপনার ফিচারগুলো ফিল্টার বা প্রসেস করার পর সেগুলো ডাউনলোড করতে পারেন।

## ১. শেপফাইল (SHP) হিসেবে এক্সপোর্ট

ArcGIS বা QGIS-এ ব্যবহারের জন্য আপনার ভেক্টর ডেটাকে **শেপফাইল (Shapefile)** হিসেবে এক্সপোর্ট করুন।

=== "JavaScript"
    ```javascript
    // এক্সপোর্ট করার জন্য কালেকশনটি ডিফাইন করুন
    var myCollection = france; // আগের উদাহরণ থেকে

    // এক্সপোর্ট করুন
    Export.table.toDrive({
      collection: myCollection,
      description: 'France_Boundary',
      fileFormat: 'SHP'
    });
    ```
    *বড় কালেকশন এক্সপোর্ট করলে এটি একাধিক ফাইলে ভাগ হয়ে যেতে পারে।*

=== "Python"
    ```python
    import geemap

    # শেপফাইল হিসেবে এক্সপোর্ট করুন
    geemap.ee_export_vector_to_drive(
        france, 
        description='France_Boundary', 
        fileFormat='SHP'
    )
    ```

## ২. CSV (টেবিল) হিসেবে এক্সপোর্ট

আপনার ফিচারগুলো যদি পয়েন্ট বা কেবল ডেটা (যেমন- শহরের তালিকা) হয়, তাহলে আপনি স্প্রেডশিট হিসেবে এক্সপোর্ট করতে পারেন।

=== "JavaScript"
    ```javascript
    Export.table.toDrive({
      collection: myCollection,
      description: 'City_List',
      fileFormat: 'CSV'
    });
    ```

=== "Python"
    ```python
    geemap.ee_export_vector_to_drive(
        myCollection,
        description='City_List',
        fileFormat='CSV'
    )
    ```
