### গুগল আর্থ ইঞ্জিনে জ্যামিতি (Geometries)

গুগল আর্থ ইঞ্জিন (GEE) স্থানিক বিশ্লেষণের জন্য বিভিন্ন জ্যামিতি প্রদান করে। নিচে জ্যামিতি ঘোষণা ও ব্যবহারের ব্যাখ্যা দেওয়া হয়েছে।

---

### **১. পয়েন্ট (Points)**

একটি **পয়েন্ট** অক্ষাংশ ও দ্রাঘিমাংশ দ্বারা সংজ্ঞায়িত একটি একক অবস্থান।

```javascript
var poi = ee.Geometry.Point(0, 45);
```

---

### **২. মাল্টি-পয়েন্ট (Multi-Points)**

```javascript
var multi = ee.Geometry.MultiPoint([0, 45, 5, 6, 70, -56]);
```

---

### **৩. লাইনস্ট্রিং (LineString)**

```javascript
var lineStr = ee.Geometry.LineString([[0, 45], [5, 6], [70, -56]]);
```

---

### **৪. মাল্টি-লাইনস্ট্রিং (Multi-LineString)**

```javascript
var mLineStr = ee.Geometry.MultiLineString(
    [[[0, 45], [5, 6], [70, -56]], [[0, -45], [-5, -6], [-70, 56]]]
);
```

---

### **৫. লিনিয়ার রিং (LinearRing)**

একটি বন্ধ লাইনস্ট্রিং, একটি লুপ গঠন করে। প্রথম ও শেষ পয়েন্ট অবশ্যই একই হতে হবে।

```javascript
var linRin = ee.Geometry.LinearRing([0, 45, 5, 6, 70, -56, 0, 45]);
```

---

### **৬. আয়তক্ষেত্র (Rectangle)**

```javascript
var rect = ee.Geometry.Rectangle(0, 0, 60, 30);
```

---

### **৭. পলিগন (Polygon)**

```javascript
var poly = ee.Geometry.Polygon([[[0, 0], [6, 3], [5, 5], [-30, 2], [0, 0]]]);
```

---

### **৮. মাল্টি-পলিগন (Multi-Polygon)**

```javascript
var multiPoly = ee.Geometry.MultiPolygon([
    ee.Geometry.Polygon([[0, 0], [6, 3], [5, 5], [-30, 2], [0, 0]]),
    ee.Geometry.Polygon([[0, 0], [-6, -3], [-5, -5], [30, -2], [0, 0]])
]);
```

### ফিচার ও ফিচার কালেকশন

* **ফিচার**: প্রপার্টি সহ জ্যামিতি (যেমন- নাম, জনসংখ্যা)।
* **ফিচার কালেকশন**: ফিচারের গ্রুপ।

```javascript
var feature = ee.Feature(ee.Geometry.Point(0, 0), {name: 'একটি পয়েন্ট'});
var featureCollection = ee.FeatureCollection([
    ee.Feature(ee.Geometry.Point(0, 0), {name: 'পয়েন্ট ১'}),
    ee.Feature(ee.Geometry.Point(1, 1), {name: 'পয়েন্ট ২'})
]);
```

### **জ্যামিতির সাধারণ অপারেশন**

১. **ক্ষেত্রফল**: `geometry.area(maxError)`
২. **দৈর্ঘ্য**: `lineString.length(maxError)`
৩. **পরিসীমা**: `geometry.perimeter(maxError)`
৪. **সরলীকরণ**: `geometry.simplify(maxError)`
৫. **কেন্দ্রবিন্দু**: `geometry.centroid(maxError)`
৬. **বাফারিং**: `geometry.buffer(radius, maxError)`
৭. **বাউন্ডিং আয়তক্ষেত্র**: `geometry.bounds(maxError)`
৮. **কনভেক্স হাল**: `geometry.convexHull(maxError)`
৯. **ছেদ**: `geometry1.intersection(geometry2, maxError)`
১০. **ইউনিয়ন**: `geometry1.union(geometry2, maxError)`
