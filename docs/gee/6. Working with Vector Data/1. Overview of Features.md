# Features (Points, Lines, Polygons)

In Earth Engine, a single shape is called a **Feature**.
It can be a point (like a city), a line (like a road), or a polygon (like a field).

## Creating Features

### 1. Point

=== "JavaScript"
    ```javascript
    // A point at [longitude, latitude]
    var point = ee.Geometry.Point([-122.2, 37.7]);

    Map.addLayer(point, {color: 'red'}, 'My Point');
    ```

=== "Python"
    ```python
    # A point at [longitude, latitude]
    point = ee.Geometry.Point([-122.2, 37.7])

    Map.addLayer(point, {'color': 'red'}, 'My Point')
    ```

### 2. LineString

A line connects multiple points.

=== "JavaScript"
    ```javascript
    var river = ee.Geometry.LineString([
      [-122.0, 37.0],
      [-122.1, 37.1],
      [-122.2, 37.2]
    ]);

    Map.addLayer(river, {color: 'blue'}, 'My River');
    ```

=== "Python"
    ```python
    river = ee.Geometry.LineString([
      [-122.0, 37.0],
      [-122.1, 37.1],
      [-122.2, 37.2]
    ])

    Map.addLayer(river, {'color': 'blue'}, 'My River')
    ```

### 3. Polygon

A polygon is a closed shape (like a rectangle or custom drawing).

=== "JavaScript"
    ```javascript
    // A rectangle defined by [minLon, minLat, maxLon, maxLat]
    var farm = ee.Geometry.Rectangle([-122.5, 37.5, -122.4, 37.6]);

    Map.addLayer(farm, {color: 'green'}, 'My Farm');
    ```

=== "Python"
    ```python
    # A rectangle defined by [minLon, minLat, maxLon, maxLat]
    farm = ee.Geometry.Rectangle([-122.5, 37.5, -122.4, 37.6])

    Map.addLayer(farm, {'color': 'green'}, 'My Farm')
    ```
