# পাইথন `id()` ফাংশন (Python id() Function)

পাইথনে, মেমোরিতে সংরক্ষিত প্রতিটি ডেটা এলিমেন্ট বা অবজেক্টকে একটি সংখ্যাসূচক মান দিয়ে চিহ্নিত করা হয়। এই মানগুলো একটি অবজেক্টকে অন্যটি থেকে আলাদা করতে সাহায্য করে। এই সংখ্যাসূচক মানকেই বলা হয় একটি অবজেক্টের **আইডেন্টিটি (Identity)**।

পাইথনের `id()` ফাংশনটি মেমোরিতে সংরক্ষিত কোনো অবজেক্টের এই অনন্য আইডেন্টিফিকেশন নাম্বারটি ফেরত দেয়। এটি অনেকটা C ল্যাঙ্গুয়েজ-এর মেমোরি অ্যাড্রেস-এর মতো।

## সংজ্ঞা (Definition)

পাইথনের `id()` ফাংশনটি একটি প্যারামিটার গ্রহণ করে এবং সেই অবজেক্টের একটি অনন্য আইডেন্টিটি (একটি পূর্ণসংখ্যা বা integer) প্রদান করে।

### এটি কীভাবে কাজ করে?

যখন আমরা কোনো অবজেক্টকে `id()` ফাংশনের ভেতরে ইনপুট হিসেবে দিই, তখন ফাংশনটি সেই অবজেক্টের জন্য একটি আইডেন্টিটি নির্ধারণ করে। এই পূর্ণসংখ্যা মানটি প্রোগ্রামের চলাকালীন সময় স্থির থাকে এবং এটি পরিবর্তন করা যায় না। প্রতিটি অবজেক্টের নিজস্ব এবং অনন্য আইডেন্টিটি থাকে।

### সিনট্যাক্স (Syntax)

```python
id(object)
```

এখানে, `object` হতে পারে ইন্টিজার, ফ্লোট, স্ট্রিং, টিউপল, লিস্ট, ক্লাস, ফাংশন ইত্যাদি।

**বিঃদ্রঃ**: `id()` ফাংশন কেবল একটি প্যারামিটার নিতে পারে।

### `id()` কেন গুরুত্বপূর্ণ?

পাইথন অবজেক্টের আইডেন্টিটি ব্যবহার করে ভ্যারিয়েবলের মানগুলো 'ক্যাশ' (cache) করে রাখতে পারে। এর ফলে পাইথনের কর্মক্ষমতা বা পারফরম্যান্স বৃদ্ধি পায়। একই মানসম্পন্ন একাধিক ভ্যারিয়েবল কখনও কখনও একই অবজেক্টকে নির্দেশ করতে পারে এবং তাদের `id()` মানও একই হতে পারে।

## সাধারণ উদাহরণ

```python
# id() ফাংশন ব্যবহারের উদাহরণ
a = 10
b = 11
text = 'Hello'

print('ID of a =', id(a))
print('ID of b =', id(b))
print('ID of text =', id(text))
```

**আউটপুট (উদাহরণস্বরূপ):**

```
ID of a = 9781088
ID of b = 9781120
ID of text = 139894857394352
```

## ইমিউটেবল অবজেক্ট (Immutable Objects)

কিছু ক্ষেত্রে পাইথন একাধিক অবজেক্টকে হুবহু একই আইডেন্টিফিকেশন নাম্বার দেয়। এটি সাধারণত ইমিউটেবল (যা পরিবর্তন করা যায় না) অবজেক্ট যেমন- ইন্টিজার, ফ্লোট, স্ট্রিং বা টিউপলের ক্ষেত্রে ঘটে। এটি মেমোরি বাঁচাতে সাহায্য করে।

```python
txt1 = 'Python'
txt2 = 'Python'

print(id(txt1))
print(id(txt2))
# দুটির আইডি একই হবে কারণ তাদের ভ্যালু একই এবং তারা ইমিউটেবল।
```

### সংক্ষেপে সারাংশ

পাইথনে `id()` ফাংশনটি মূলত মেমোরি ম্যানেজমেন্ট এবং অবজেক্ট আইডেন্টিটি নিশ্চিত করার জন্য ব্যবহৃত হয়। এটি ডেভেলপারদের বুঝতে সাহায্য করে যে দুটি ভিন্ন ভ্যারিয়েবল কি আসলে মেমোরির একই জায়গাকে নির্দেশ করছে কি না।
