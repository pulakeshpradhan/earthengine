# Data Type Comparison

| Google Earth Engine | Python | JSON | Example |
| :--- | :--- | :--- | :--- |
| `ee.Dictionary` | `dict` | `object` | GEE: `ee.Dictionary({"name": "John"})`, Python: `{"name": "John"}`, JSON: `{"name": "John"}` |
| `ee.List` | `list` | `array` | GEE: `ee.List([1, 2, 3])`, Python: `[1, 2, 3]`, JSON: `[1, 2, 3]` |
| `ee.String` | `str` | `string` | GEE: `ee.String("hello")`, Python: `"hello"`, JSON: `"hello"` |
| `ee.Number` (Int) | `int` | `number` | GEE: `ee.Number(123)`, Python: `123`, JSON: `123` |
| `ee.Number` (Float) | `float` | `number` | GEE: `ee.Number(3.14)`, Python: `3.14`, JSON: `3.14` |
| `true` | `True` | `true` | GEE: `true`, Python: `True`, JSON: `true` |
| `false` | `False` | `false` | GEE: `false`, Python: `False`, JSON: `false` |
| `ee.Null` | `None` | `null` | GEE: `ee.Null`, Python: `None`, JSON: `null` |

## Detailed Examples

### Strings

=== "JSON"
    ```json
    "Hello, World!"
    ```
=== "Python"
    ```python
    "Hello, World!"
    ```
=== "GEE"
    ```javascript
    ee.String("Hello, World!")
    ```

### Numbers / Integers

=== "JSON"
    ```json
    123
    ```
=== "Python"
    ```python
    123
    ```
=== "GEE"
    ```javascript
    ee.Number(123)
    ```

=== "JSON (Float)"
    ```json
    3.14
    ```
=== "Python (Float)"
    ```python
    3.14
    ```
=== "GEE (Float)"
    ```javascript
    ee.Number(3.14)
    ```

### Lists

=== "JSON"
    ```json
    [1, 2, 3, 4, 5]
    ```
=== "Python"
    ```python
    [1, 2, 3, 4, 5]
    ```
=== "GEE"
    ```javascript
    ee.List([1, 2, 3, 4, 5])
    ```

### Arrays

* JSON: `[1, 2, 3, 4, 5]`
* Python: `array([1, 2, 3, 4, 5])` or `numpy.array([1, 2, 3, 4, 5])`
* GEE: `ee.Array([1, 2, 3, 4, 5])`

### Multidimensional Arrays

* JSON: `[[1, 2], [3, 4], [5, 6]]`
* Python: `array([[1, 2], [3, 4], [5, 6]])` or `numpy.array([[1, 2], [3, 4], [5, 6]])`
* GEE: `ee.Array([[1, 2], [3, 4], [5, 6]])`

### Object/Dictionaries

* JSON: `{name: "John", age: 30}`
* Python: `{"name": "John", "age": 30}`
* GEE: `ee.Dictionary({"name": "John", "age": 30})`

### Booleans

* JSON: `true`
* Python: `True`
* GEE: `true`

* JSON: `false`
* Python: `False`
* GEE: `false`

### Null Values

* JSON: `null`
* Python: `None`
* GEE: `ee.Null`

## Additional Google Earth Engine Data Types

### Date

* GEE: `var date = ee.Date('2022-01-01')`

### Image

* GEE: `var image = ee.Image('LANDSAT/LC08/C02/T1_TOA/LC08_044034_20220101')`

### Image Collection

* GEE: `var ic = ee.ImageCollection('LANDSAT/LC08/C02/T1_TOA')`

### Feature

* GEE: `var feature = ee.Feature(ee.Geometry.Point([-114.318, 38.985]));`

### Feature Collection

GEE: `var fc = ee.FeatureCollection('WRI/GPPD/power_plants');`

### Geometry

* Point: `var geometry = ee.Geometry.Point([-121.90000864115225, 37.46578611849061]);`
* LineString: `var geometry = ee.Geometry.LineString( [[-121.91030832376944, 37.46687611458185], [-121.87992426004874, 37.47818388623988], [-121.84130045023429, 37.46115345781038]]);`
* Polygon: `var geometry =  ee.Geometry.Polygon( [[[-121.90739008036124, 37.479409927344875], [-121.90739008036124, 37.447390037753415], [-121.85383173075186, 37.447390037753415], [-121.85383173075186, 37.479409927344875]]], null, false);`
