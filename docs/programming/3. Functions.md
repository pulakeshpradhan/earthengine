# Working with Functions (Custom Tools)

A **Function** is like a mini-recipe.
Instead of writing "Mix flour, sugar, and eggs" every time, you create a function called `MakeDough()`.

## Why use functions?

They save time! Instead of repeating yourself, you write the logic once and reuse it.

## Example: A Simple Calculator

Let's make a function that adds two numbers.

=== "JavaScript"
    ```javascript
    // Define the function
    var addNumbers = function(a, b) {
      return a + b;
    };

    // Use it!
    var result = addNumbers(5, 10);
    print(result); // Output: 15
    ```

=== "Python"
    ```python
    # Define the function
    def add_numbers(a, b):
      return a + b

    # Use it!
    result = add_numbers(5, 10)
    print(result) # Output: 15
    ```

## Example: A Cloud Masking Function (Real World)

In Earth Engine, we use functions mostly to **hide clouds**.
This function takes an image with clouds, and returns a clean image.

=== "JavaScript"
    ```javascript
    // Define the function
    var maskClouds = function(image) {
      // Find the clouds using the QA band
      var qa = image.select('QA_PIXEL');
      var cloudMask = qa.bitwiseAnd(1 << 3).eq(0); // Complex logic hidden here!

      // Update the mask (hide clouds)
      return image.updateMask(cloudMask);
    };

    ```

=== "Python"
    ```python
    # Define the function
    def mask_clouds(image):
    ```
