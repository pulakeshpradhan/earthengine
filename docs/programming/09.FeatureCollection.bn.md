# গুগল আর্থ ইঞ্জিন-এ ফিচার কালেকশন (FeatureCollection in GEE)

এই উদাহরণের মাধ্যমে আমরা `FeatureCollection`-এর বিভিন্ন কাজ যেমন ফিল্টারিং (filtering), ম্যাপিং (mapping) এবং এগ্রিগেটিং (aggregating) সম্পর্কে জানব।

## ফিচার কালেকশন লোড করা (Loading a FeatureCollection)

```javascript
// বিভিন্ন দেশের সীমানা সম্বলিত ফিচার কালেকশন লোড করুন
var countries = ee.FeatureCollection('USDOS/LSIB_SIMPLE/2017');
print(countries);
print(countries.first());
```

এখানে আমরা দেশগুলোর সীমানা সম্বলিত একটি কালেকশন লোড করেছি। `print(countries)` পুরো কালেকশনের মেটাডেটা দেখায় এবং `print(countries.first())` কালেকশনের প্রথম ফিচারটি দেখায়।

## ফিচার কালেকশনের ওপর ম্যাপিং (Mapping over a FeatureCollection)

```javascript
// প্রতিটি ফিচারের নাম সংগ্রহ করার জন্য একটি ফাংশন ব্যবহার করুন 
var names = countries.map(function(x) {
  // কেবল 'country_na' প্রোপার্টিসহ একটি নতুন ফিচার রিটার্ন করুন
  return ee.Feature(null, {'country_na': x.get('country_na')});
});
print(names);
```

এটি দেখায় কীভাবে `.map()` ফাংশন ব্যবহার করে কালেকশনের প্রতিটি ফিচারের ওপর কোনো সুনির্দিষ্ট কাজ করা যায়।

## ডেটা এগ্রিগেট করা (Aggregating Data)

```javascript
// ফলাফলকে একটি লিস্টে রূপান্তর করুন
var list = names.aggregate_array('country_na');
// লিস্টটি প্রিন্ট করুন
print(list, 'Countries List');
```

`aggregate_array` ফাংশনটি প্রতিটি ফিচার থেকে একটি নির্দিষ্ট প্রোপার্টি সংগ্রহ করে এবং সেগুলোকে একটি তালিকায় বা লিস্টে একত্রিত করে।

## ফিচার কালেকশন ফিল্টার করা (Filtering)

```javascript
// কেবল 'Nepal' নামযুক্ত দেশটি নির্বাচন করুন
var nepal = countries.filter(ee.Filter.eq('country_na', 'Nepal'));
Map.addLayer(nepal);
```

এটি দেখায় কীভাবে `ee.Filter` ব্যবহার করে বিশাল কালেকশন থেকে আপনার প্রয়োজনীয় ডেটা খুঁজে বের করবেন।

## মূল ধারণাসমূহ (Key Concepts)

১. **লোড করা**: `ee.FeatureCollection()` এর মাধ্যমে ডেটা লোড করা।
২. **নিরীক্ষণ**: `print()` এর মাধ্যমে ফিচারগুলো পরীক্ষা করা।
৩. **ম্যাপিং**: `map()` ব্যবহার করে সব ফিচারের ওপর একই নিয়ম প্রয়োগ করা।
৪. **এগ্রিগেট**: `aggregate_array()` ব্যবহার করে সব ফিচারের মান এক জায়গায় করা।
৫. **ফিল্টারিং**: নির্দিষ্ট শর্তানুযায়ী ডেটা বেছে নেওয়া।
৬. **ভিজ্যুয়ালাইজেশন**: ম্যাপে ফলাফল প্রদর্শন করা।

## উপসংহার

ফিচার কালেকশন হলো আর্থ ইঞ্জিনের অত্যন্ত প্রয়োজনীয় একটি মাধ্যম যার সাহায্যে আপনি দেশের সীমানা, পয়েন্ট অফ ইন্টারেস্ট বা যেকোনো ভেক্টর ডেটা নিয়ে কাজ করতে পারেন।
