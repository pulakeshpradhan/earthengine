# গুগল আর্থ ইঞ্জিন-এ ডিকশনারি (Dictionaries in GEE)

গুগল আর্থ ইঞ্জিন-এ **ডিকশনারি (Dictionaries)** হলো কি-ভ্যালু পেয়ার (key-value pair) ভিত্তিক ডেটা স্ট্রাকচার যা ডেটা সংরক্ষণ ও সুসংগঠিত করতে ব্যবহৃত হয়। এগুলো `ee.Dictionary` অবজেক্ট হিসেবে বাস্তবায়িত এবং জাভাস্ক্রিপ্ট অবজেক্ট বা পাইথন ডিকশনারির মতোই কাজ করে।

## ডিকশনারি তৈরি করা (Creating Dictionaries)

### ১. `ee.Dictionary()` ব্যবহার করে

```javascript
var dict = ee.Dictionary({
  'name': 'John Doe',
  'age': 30,
  'city': 'New York'
});
print('Dictionary:', dict);
```

### ২. কী এবং ভ্যালুর লিস্ট থেকে

```javascript
var keys = ['a', 'b', 'c'];
var values = [1, 2, 3];
var dictFromLists = ee.Dictionary.fromLists(keys, values);
print('Dictionary from lists:', dictFromLists);
```

## ডিকশনারির ভ্যালু অ্যাক্সেস করা (Accessing Values)

### ১. get() ব্যবহার করে

```javascript
var name = dict.get('name');
print('Name:', name);
```

### ২. সুনির্দিষ্ট টাইপ অনুযায়ী অ্যাক্সেস

```javascript
var age = dict.getNumber('age'); // সংখ্যা হিসেবে পেতে
print('Age as number:', age);
```

## ডিকশনারি পরিবর্তন করা (Modifying Dictionaries)

গুগল আর্থ ইঞ্জিনের ডিকশনারিগুলো ইমিউটেবল (immutable) বা অপরিবর্তনীয়। তবে আপনি চাইলে পরিবর্তনসহ নতুন ডিকশনারি তৈরি করতে পারেন।

### ১. নতুন কি-ভ্যালু যুক্ত বা আপডেট করা

```javascript
var updatedDict = dict.set('occupation', 'Engineer');
print('Updated Dictionary:', updatedDict);
```

### ২. কোনো কি-ভ্যালু সরিয়ে ফেলা

```javascript
var reducedDict = dict.remove(['age']);
print('Dictionary with age removed:', reducedDict);
```

## ডিকশনারি অপারেশনসমূহ (Dictionary Operations)

### ১. একাধিক ডিকশনারি একত্রিত করা

```javascript
var dict1 = ee.Dictionary({'a': 1, 'b': 2});
var dict2 = ee.Dictionary({'c': 3, 'd': 4});
var combinedDict = dict1.combine(dict2);
print('Combined Dictionary:', combinedDict);
```

### ২. কোনো কি (key) আছে কিনা যাচাই করা

```javascript
var hasKey = dict.contains('name');
print('Dictionary contains "name":', hasKey);
```

## উন্নত ব্যবহার (Advanced Usage)

### ইমেজের প্রোপার্টিকে ডিকশনারিতে রূপান্তর

```javascript
var image = ee.Image('LANDSAT/LC08/C02/T1_L2/LC09_044034_20230215');
var properties = image.toDictionary();
print('Image properties as dictionary:', properties);
```

## সেরা অনুশীলনসমূহ (Best Practices)

১. **বর্ণনামূলক নাম**: সহজবোধ্যতার জন্য সবসময় বর্ণনামূলক 'কী' (key) নাম ব্যবহার করুন।
২. **অপরিবর্তনীয়তা**: মনে রাখবেন GEE-তে ডিকশনারিগুলো ইমিউটেবল।
৩. **সঠিক গেটার মেথড**: যখন আপনি ভ্যালুর ধরণ (যেমন- স্ট্রিং বা নাম্বার) জানেন, তখন `getString()` বা `getNumber()` মেথড ব্যবহার করা ভালো।

## উপসংহার

গুগল আর্থ ইঞ্চিন-এ জটিল ইমেজ এবং ফিচারের প্রোপার্টি ম্যানেজ করতে ডিকশনারি একটি অত্যন্ত কার্যকরী ডেটা স্ট্রাকচার।
