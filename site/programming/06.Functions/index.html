    Functions in Google Earth Engine¶

    =====================================

    Functions in Google Earth Engine (GEE) are essential for creating reusable code and performing complex operations. They can be used to manipulate images, features, and other Earth Engine objects.

    Types of Functions in GEE¶

    1. Built-in Functions¶

    Provided by the Earth Engine API.

    2. User-defined Functions¶

    Created by users for custom operations.

    Creating and Using Functions¶

    Basic Function¶

    // Define a function to calculate NDVI
var calculateNDVI = function(image) {
  return image.normalizedDifference(['NIR', 'RED']).rename('NDVI');
};

// Use the function
var landsat8 = ee.ImageCollection('LANDSAT/LC08/C02/T1_L2')
  .filterDate('2022-01-01', '2022-12-31');
var ndviCollection = landsat8.map(calculateNDVI);

    Functions with Multiple Arguments¶

    // Function to calculate a custom index
var calculateIndex = function(image, band1, band2, newName) {
  return image.normalizedDifference([band1, band2]).rename(newName);
};
    // Use the function var customIndex = landsat8.map(function(image) { return calculateIndex(image, 'SR_B5', 'SR_B4', 'CustomIndex'); });

    Anonymous Functions (Lambda Functions)¶

    // Use an anonymous function directly in map()
var ndviCollection = landsat8.map(function(image) {
  return image.normalizedDifference(['SR_B5', 'SR_B4']).rename('NDVI');
});

    Earth Engine Algorithms as Functions¶

    Many Earth Engine algorithms can be used as functions:¶

    // Use ee.Algorithms.Landsat.simpleComposite() as a function
var composite = ee.Algorithms.Landsat.simpleComposite({
  collection: landsat8,
  asFloat: true
});

    Higher-Order Functions¶

    GEE supports higher-order functions like map(), reduce(), and iterate():¶

    // Using reduce() to get the mean NDVI
var meanNDVI = ndviCollection.select('NDVI').reduce(ee.Reducer.mean());

// Using iterate() to accumulate area
var features = ee.FeatureCollection('TIGER/2018/States');
var totalArea = features.iterate(function(feature, sum) {
  return ee.Number(sum).add(feature.geometry().area());
}, ee.Number(0));

    Best Practices¶

    1. Focus on a Single Task Keep functions focused on a single task.
    2. Use Descriptive Names Use descriptive names for functions and arguments.
    3. Comment Your Code Comment your code to explain complex operations.
    4. Reuse Functions Reuse functions to keep your code DRY (Don't Repeat Yourself). Conclusion

    Functions in Google Earth Engine are powerful tools for creating efficient, reusable, and maintainable code for geospatial analysis and remote sensing tasks. By mastering functions, you can streamline your workflows, simplify complex operations, and focus on extracting valuable insights from Earth observation data.
